# Original Description
This changes the allocation strategy of `byte_stream` to something closer to other libraries - the buffer capacity is doubled at minimum each time more space is needed. This lowers the number of calls needed to grow the buffer size while serializing data.

There is another patch incoming that allows for the buffer to be `realloc`ed smaller when serialization is completed.
