# Original Description
Some users use a long/strong password in order to protect the wallet file. However, it's annoying having to type a strong password every time a transaction is made, specially when a lot of transactions are sent in a short period of time.

I suggest adding a new option "Ask for password before sending a transaction" in Settings > Interface page.
