# Original Description
While creating/sending a transaction in the GUI, it should warn with a pop-up box about the inability to send a transaction due to insufficient (unlocked) funds.

However, on some of my builds I do not receive these warnings/pop-ups and the GUI just crashes.

This PR could be merged if other people experience this. The fix is to render the 'Send' button inactive on insufficient unlocked balance.
