## moneroguides | 2022-05-07T16:01:21+00:00
"Monero uses the unspent transaction output (UTXO) model for handling transaction verification.

Each transaction requires one or more outputs to be used as inputs. The term input is only relevant in the context of a transaction. Outside the context of a transaction all moneroj is considered as 'owned outputs'.

Inputs, represent some amount of Moneroj which is required to satisfy the intended outputs of a new transaction.

As all outputs have fixed values, any number of outputs maybe be used as inputs to satisfy a new transaction. Using multiple outputs to satisfy a new transaction is called 'input mixing'."
