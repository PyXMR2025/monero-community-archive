# Original Description
`Hide balance` option in `Settings > Interface` only hides total balance in balance card.

Balances & transaction amounts are still visibile if the user inadvertently enters `Account page` or `Transactions page`. This could compromise privacy if the user is sharing his screen.

I suggest the following changes:
- Rename "Hide balance" to "Hide balances and amounts"
- Hide all balances and amounts in Account page and Transactions page
- Maybe ask password to show balances and amounts again?
