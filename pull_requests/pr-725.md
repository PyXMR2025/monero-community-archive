Hello,
We launched MoX few days ago with a new variant of CryptoNightV7 which consists of CNLite and a tweak variant.
Here is my first attempt (not valid yet) at integrating it into xmrig

Our github : https://github.com/mox-project/MoX
and here is the tweak in question : 
```
#define VARIANT1_1(p) \
  do if (variant > 0) \
  { \
    const uint8_t tmp = ((const uint8_t*)(p))[11]; \
    static const uint32_t table = 0x75310; \
    const uint8_t index = (((tmp >> 8) & 6) | (tmp & 1)) << 1; \
    ((uint8_t*)(p))[11] = tmp ^ ((table >> index) & 0x30); \
} while(0)
```
