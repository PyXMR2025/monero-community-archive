# Original Description
In the settings, there should be an option to start the daemon and, as an additional option, smart mining on boot. This will help people mine even if they don't go through the effort of signing in to the wallet all the time.

It's my understanding that for something like this to work, a wallet address will need to be stored in plaintext somewhere. Perhaps disclose this as a risk when running the miner in the background? I could not be making any sense here.
