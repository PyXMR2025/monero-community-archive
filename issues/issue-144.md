# Original Description
# Tab sizes
I think these could be sized better. The 'Amount' tab being too wide for example. I would say that if possible to size them to fit their content.

# Order toggles
I think these add visual noise to the feature. It would be nice to only have one arrow per tab that alternates up/down based on the state. Also to hide the arrow when the tab isn't active.

This might help with the minimised view as I noticed that the 'Amount' tab is off-screen.
