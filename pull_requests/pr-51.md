# Original Description
NOTE : Please test build before merging. I couldn't test on Windows platforms.

This is an high level Wallet class for Monero. Created as a "wallet2.h" wrapper.
The goal is to provide easier methods and data structures than raw "wallet2.h" class.

It could be used as lib in the future (pools, exchanges ?).
Run "doxygen Doxyfile" for complete documentation.

Important notes :
- Tested manually in a custom program + partially integrated to bitmonero-qt, BUT no functional tests are available yet (this need to be done asap). I will try to work with existing tests.
- Added to CMake as shared library. I don't know either if it is a good choice or not, and I couldn't test it on Windows/MAC platforms.

If you find missing methods (or weird things), please comment.
