# Original Description
The Trusted Node checkbox is not actually reflecting the right value. I have had the box unchecked and been able to rescan the balance, and I've had it checked and it is saying the remote node is untrusted.

This is a pretty big hole that could compromise the privacy of a user who may think they have unchecked a remote node as untrusted but are actually treating it as trusted.
