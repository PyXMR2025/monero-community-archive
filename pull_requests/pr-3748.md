# Original Description
I was having difficulty getting the monerod.service to start on boot following the instructions provided. The instructions were missing a step on enabling the service.

I believe the correct install location for the service file should be in /lib/systemd/system, not /etc/systemd/system as was originally stated. The service is then enabled using systemctl enable monerod.service which creates a symbolic link in /etc/systemd/system to the service file installed in /lib/systemd/system.

At least that is how it is on my Linux Mint 18.3 system. The service is now starting at bootup.
