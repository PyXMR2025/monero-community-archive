# Original Description
To make sure one can use monero like digital cash the transaction history must be deletable.
Currently when someone gets hold of your Seed they can see when and in which amount you used monero. Setting up a new wallet means you have to make sure all your friends and previous transaktionen partners now use the new address. To make monero conveniently usable i suggest the view key can be changed without getting a new seed and new address. So you can delete the old view key and therefore hide the transaction history.
