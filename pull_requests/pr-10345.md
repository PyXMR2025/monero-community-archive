# Original Description
Builds on top of:
- #10342
- #10111
- #10135

When inserting outputs into the FCMP++ tree, we convert each output pubkey, key image generator, and commitment (ed25519 points) into Wei x and y coordinates (as part of the [tower cycle](https://gist.github.com/tevador/4524c2092178df08996487d4e272b096) critical for the curve tree -- Wei coordinates are "Selene" scalars).

This PR introduces the crypto functions necessary to perform the conversion:

- `point_to_ed_derivatives`
- `ed_derivatives_to_wei_x_y`
- `fe_ed_derivatives_to_wei_x_y`

It uses the equation from E.2 in https://www.ietf.org/archive/id/draft-ietf-lwig-curve-representations-02.pdf.
