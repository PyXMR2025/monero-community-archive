# Original Description
Changed Blockchain::for_all_blocks() to for_blocks_range()
Operate on blockchain in-place instead of building a copy first.

Trying to use print_coinbase_tx_sum on the entire blockchain would kill my daemon on VPS with out of memory because it would try to build an in-memory copy of the entire blockchain before processing. Now the command still takes a long time to run but it actually finishes instead of killing the server.
