## 0xFFFC0000 | 2025-11-24T13:17:37+00:00
The build failure is caused by an outdated CMake detection heuristic in Monero’s CMakeLists.txt that’s incorrectly setting the -Dstatic_assert=_Static_assert compile definition. This macro interferes with the built-in static_assert keyword in modern C++ (including C++17, which your build uses), treating it as an undeclared function call inside standard library headers like <new> and <type_traits>.
