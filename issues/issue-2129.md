# Original Description
Sometimes, when calling the `getblocktemplate` method via a curl, no answer is given and the call blocks.

Inspecting the daemon logs only shows a series of:

    2017-06-28 08:25:36.345 [RPC0]  ERROR   daemon.rpc      src/rpc/core_rpc_server.cpp:923 Failed to calculate offset for

Notes:

- I've seen this happen when calling the method several times in a short time frame (say 3 or 4 times in 60 seconds).
- I'm calling the method with `"reserve_size": 0`
