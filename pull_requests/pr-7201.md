# Original Description
I felt like the different texts the `help` command can output were inconsistently formatted. Thus I unified the formatting to the rules that seemed to be obeyed the most. I did the following:
- Add newline characters into long texts.
- Add missing "." to the end of sentences.
- Remove trailing newline characters.
- Remove extraneous indentation (for `help set` and `help mms set`).
- Fix wrong indentation in the code.
- Always use double quotes within texts instead of single quotes.

If you feel like I went too far, please let me now and I'll take some changes back.
