# Version: v0.17.1.7

# Release Notes
# Overview

This is the v0.17.1.7 minor point release of the Monero GUI software.
This is a recommended release that contains P2P network layer improvements.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.17.1.7)

Some highlights of this minor release are:

- Ask for writing desktop shortcut on first start (Linux)
- Fix wallet initialization flag handling
- Get back "Sending transaction ..." splash
- Disable QML cache
- Minor bug fixes
