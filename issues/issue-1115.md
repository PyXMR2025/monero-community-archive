# Original Description
Currently sweep_all only works with an address. That is, `sweep_all <address>`. However, people might want to sweep their wallet to an exchange, which requires a payment ID. Therefore, it would be convenient if people were also able to specify a payment ID with this feature. Thus:

`sweep_all <address> <payment ID>`
