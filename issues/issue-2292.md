# Original Description
**Describe the bug**
More of a feature. While XMRIG does support SOCKS5, there is no way to use a proxy that requires logging in
It would be great to be able to use `username:password@proxy-ip:port` format to connect to a protected proxy
Few VPN providers give us SOCKS5 profiles which can be used to bypass network restrictions.


**Required data**
 - OS: ALL
