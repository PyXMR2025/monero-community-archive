# Original Description
![](https://i.redd.it/1wcbs2rfk4xz.png)

Author: knueffelbund
Date: 10/11/2017
Reddit thread: [link](https://www.reddit.com/r/Monero/comments/7c0bo7/had_some_fun_yesterday_evening_restyling_the/)
[zpl.io design link](https://zpl.io/2EmDOKD) (ask for invite)

___
Mostly a change of colors, except for:

- Restyled balance information
- Larger titlebar + logo
- Smaller menu buttons
- Change of menu button indicators
- Optional icon placement in buttons

As such, more trivial to implement than [UI Proposal #0](https://github.com/monero-project/monero-core/issues/419), [UI Proposal #1](https://github.com/monero-project/monero-core/issues/869) for the short-term.

___
Benefits from the current design:

- A black look embodies the Monero philosophy; stealth/privacy
- Less clutter in the menu area

An optional color scheme could be made to allow for switching between light/dark mode.
