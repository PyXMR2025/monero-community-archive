# Original Description
The invoke function in the levin handler is currently unused. It is probably best to remove it because:
  * using it and `async_invoke` at the same time is going to be problematic on the ordering of responses
  * It blocks a thread waiting for a response, and requires another thread to process the response

#8569 removes related code that is also unused.
