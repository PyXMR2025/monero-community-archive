# Original Description
CPU only for now because elliptic curve operations are required for mining.

To test:
- Compile https://git.wownero.com/wownero/wownero/src/branch/dev-v0.10
- start `wownerod --testnet --offline --fixed-difficulty 10000`
- wownerod --start-mining ADDRESS --spendkey SPENDKEY
- wait until it mines past block height 60
- `stop_mining` in wownerod
- run `xmrig -o 127.0.0.1:11181 --coin wownero -u ADDRESS --spend-secret-key SPENDKEY --daemon`
