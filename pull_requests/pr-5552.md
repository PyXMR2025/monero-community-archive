# Original Description
- This addresses https://www.openssl.org/news/secadv/20190226.txt (CVE: 2019-1559) which impacted all versions of openssl-1.0.

Note that this does not address CVE-2019-1543 (https://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2019-1543), which impacts all versions of openssl 1.1 through 1.1.0j and 1.1.1b ... I do not believe monero uses any of this functionality from openssl, and 1543 does not affect 1.0.2r.

The above is patched in openssl, where it was marked as low severity.  Similar issues possibly present in monero (unrelated to openssl, directly), should be looked into w.r.t. the specific issue present in CVE-2019-1543.
