# Original Description
At the moment the GUI seems to be unusable for users using screen readers. Would be good to have at least the minimal functionalities fully accessible to blind/visually impaired people.

An issue was initially opened by a blind user in https://github.com/monero-project/monero-site/issues/1475. I redirected them here, but they didn't open an issue yet. I think it's a problem that it's important to solve regardless, so i'm opening this issue to keep track of the progresses on this front.
