# Original Description
Updated Polish translation for the upcoming v0.12.1 release as mentioned in #1377.

If you want to review the translations, you can start with ctrl+f "blackball". I used "Zablokowany" for "Blackballed" and "Odblokowany" for "Unblackballed". Blackballing probably refers to [this](https://lab.getmonero.org/pubs/MRL-0001.pdf) paper (as mentioned by **xmrscott**). You can ctrl+f "marble" in the paper for more info. Basically blackballing means not allowing an output to be used in ring signature. Maybe someone has a better translation. Imo, even in English blackballing is a bad phrase. I'd just use Block/Unblock or Blacklist/Unblacklist.

Also, I don't feel that my translations with "heights" are optimal. So you can ctrl+f "height" as well. E.g. "restore height", "wallet creation height", etc.
