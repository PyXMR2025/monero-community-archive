# Original Description
There should be a way to verify the mnemonic seed you have written down for a wallet, to ensure that it is a functioning backup. This could be done like in the Mycelium wallet: Instead of having to type out the entire seed, the user is shown multiple possible words each, and simply clicks the ones he has written down.

IMO this should be a skippable step after being shown the seed when creating a wallet, and also an option in the settings to do this at any time.
