# Original Description
We pin Boost to 1.86.0 on the release branch for rolling release distributions because of a build issue with Boost 1.87. ICU (a dependency for Boost Locale) was updated to v76 in MSYS2 causing a [linking issue](https://github.com/monero-project/monero/actions/runs/12918551559/job/36027291926#step:6:13034). This PR temporarily downgrades ICU to fix the build.

We can remove this once we have backported #9628 to the release branch. This requires updating Boost (#9687), which needs careful testing and so should not be rushed out ahead of the upcoming tag.
