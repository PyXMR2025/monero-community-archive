# Original Description
Enables making parallel requests for a small number of blocks. When scanning the chain using parallel requests, setting this request param to a small number (e.g. 10) can speed up progress feedback to the user significantly (compared to leaving the default to 1000 as is).

Used in the Seraphis lib async scanner: https://github.com/UkoeHB/monero/pull/23
