# Original Description
So I've added a verbosity levels option `-b` (or `--verbosity`) which shows the specific amount of logs.

If this option is set to 0, then no log is showed. If it's set to 1, only the configurations are shown (versions, huge pages, cpu, cpu memory, threads and pools). Otherwise, when it's set to 2, everything is shown as in previous versions of xmrig; so not only the configurations appears but also the default timed logs which give information about the time of the logs, the status of the pools, the hashrate, etc.

The default verbosity is set to 2. Everything is working so far.

Can this be added?
