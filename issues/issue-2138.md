# Discussion History
## xmrig | 2021-02-27T08:27:05+00:00
Minimum supported CentOS for compile from source is 7 and it is already too old. For this OS you can use [official static build](https://github.com/xmrig/xmrig/releases/download/v6.9.0/xmrig-6.9.0-linux-static-x64.tar.gz)  or build static executable on [Alpine](https://xmrig.com/docs/miner/build/alpine).
