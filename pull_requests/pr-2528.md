# Original Description
Given that we have an Esperanto mnemonics word list, it doesn't hurt to add a Lojban one. :)

The list is composed of all current Lojban gismu (root words) plus lojvo (compound words) based on the word usage frequency list. In total it is 1626 words with 4-letter unique prefix.

Lojban is a "regular" language so gismu are always 5-letter long, and Lojban people are good at distinguish them. Also `'` is a valid Lojban alphabet.

Sources:
* lo gimste jo'u lo ma'oste (http://guskant.github.io/lojbo/gismu-cmavo.html)
* N-grams of Lojban corpus (https://mw.lojban.org/papri/N-grams_of_Lojban_corpus)
