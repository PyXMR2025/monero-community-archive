# Original Description
Additions allowing build on NetBSD

* NetBSD's system compiler does not support `-march=native`. Extra lines in top-level _CMakeLists.txt_ allow `-march=native` to be replaced by `-mtune=native` where the former is unsupported.

* NetBSD needs an extra CFLAG ([-D_NETBSD_SOURCE](http://cdn.netbsd.org/pub/pkgsrc/current/pkgsrc/net/miniupnpc/patches/patch-Makefile)) for miniupnc. Provided in _external/CMakeLists.txt_.

* NetBSD needs an extra CFLAG ([-D_SEM_SEMUN_UNDEFINED](http://cdn.netbsd.org/pub/pkgsrc/current/pkgsrc/databases/lmdb/Makefile)) for lmdb. Provided in _external/db_drivers/CMakeLists.txt_.

* OpenBSD clauses in _external/easylogging++/easylogging++.h_ seem to be OK for NetBSD too. Replicated accordingly.

* OpenBSD clauses in _src/crypto/oaes_lib.c_ seem to be OK for NetBSD too. Replicated accordingly. Likewise (also in _src/crypto_) for _slow-hash.c_ and _tree-hash.c_.
