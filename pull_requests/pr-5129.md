I just finished to configure Pootle to host the translations for the CLI: https://translate.getmonero.org/projects/CLI/.

From now on all translation-related work should be uploaded on Pootle and Pull Requests editing translation files should be labelled as invalid and closed (@moneromooo-monero @fluffypony ).

I already added on Pootle all languages available for the GUI. After a language will be completed for at least 70/80% i will open a PR to add that language to the list (using @monero-pootle).
