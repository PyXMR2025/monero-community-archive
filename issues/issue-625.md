# Original Description
I've created a very simple HTML monitor that I believe could be "native" on XMRig, since it doesn't require any extra installation at all. Files are separated to become easier to maintain the code, but it's very possible to add everything in a single HTML file that could come with the miner if that's something desirable.

In any other case, this might be a cool thing for anyone who wants a simple hashrate monitor via web (also useful for remote monitoring). You can see some screenshots on how it looks in the GitHub repository.

**xmrig-monitor**: https://github.com/curumimxara/xmrig-monitor

I recently switched from XMR-Stak to XMRig and I'm very happy with the results. Only thing that was missing for me was the web monitor that comes with Stak so I decided to make one myself. Let me know if anyone has issues using it. Cheers.
