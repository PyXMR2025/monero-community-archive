# Original Description
Readline is now compiled with ncurses as a backend. I also attempted to build with termcap instead of a full ncurses installation, but since the releases so far have not supported readline on windows and
the termap Makefile does not handle installation directories properly that are demanded by depends,
Besides adding readline, this pull request also adds some debug targets to the depends Makefile. To check all the available targets now, you can run `cat Makefile | awk '!/SHELL/ && /^[A-z]/ {print $1}' | awk '{print substr($0, 1, length($0)-1)}' .
To speedup the depends cached builds, remove some some clutter from the package files. This mainly includes removing all the shared libraries and .la linker files. It also gives stronger guarantees that monero only links the static libs without any external rvalues.
Packages can now also be built individually and for each stage. This allows easier debugging.
The travis build fail currently with this build, but I am not sure why.
