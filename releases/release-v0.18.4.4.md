# Version: v0.18.4.4

# Release Notes
# Overview

This is the v0.18.4.4 release of the Monero GUI software. This is a recommended release that fixes a bug with Ledger hardware wallet when rejecting secret view key export.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.18.4.4)

Some highlights of this release are:

- Fix Ledger hardware wallet bug when rejecting secret view key export
- Update P2Pool to v4.12 (#4519)
- Minor bug fixes
