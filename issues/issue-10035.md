# Original Description
In several occurrences of https://github.com/monero-project/monero/blob/master/src/device/device_ledger.cpp the return values of `exchange()` and `exchange_wait_on_input()` are not verified even if they do have the return values.

E.g. for `get_secret_keys() -> send_simple() -> exchange_wait_on_input()` call stack it leads to a situation when user rejects to export the key, but the wallet is still created by the Monero GUI. 

Linked to https://github.com/monero-project/monero/issues/5361
