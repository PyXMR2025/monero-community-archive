# Original Description
This PR adds the broadcast svc and the stream impl that gets given to the connection task.

Broadcasting is handled differently to normal messages, it has a custom interface to connections instead of going through the client service. This allowa broadcasting to all peers even if they are not currently in the ClientPool.
