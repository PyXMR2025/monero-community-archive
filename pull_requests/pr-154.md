Adds the skeleton for Cuprate's user book: the end-user documentation for Cuprate, aka, "how to use `cuprated`".

Cramming everything into the architecture book doesn't make sense since 95% of it doesn't matter in a practical way for our users, so that stuff will go into this book (among other things), e.g. instrumentation, configuration, etc.
