# Original Description
## Developer Guide - Daemon RPC - Full Update
### Status
Reviewed
### Description
This PR solves #709
Actually, the Daemon RPC Developer Guide is outdated.
I'm trying to add everything that is actually missing:
- Review each call
- Add missing calls
- Add new inputs/outputs
- Add descriptions
- Update examples
### Working process
Here is my process for doing this:
1. Check the RPC calls (JSON first) from the [Core RPC Server](https://github.com/monero-project/monero/blob/master/src/rpc/core_rpc_server.h) file.
2. For each RPC call, look it up in the [Command Definition](https://github.com/monero-project/monero/blob/master/src/rpc/core_rpc_server_commands_defs.h) file.
3. Add missing information to the Developer Guide:
    1. For a new RPC Call, add it in the Guide, at the same position it is in the list from [Core RPC Server](https://github.com/monero-project/monero/blob/master/src/rpc/core_rpc_server.h) file.
    2. For an existing Call:
        1. Get the new RPC output example
        2. Add the new inputs/outputs in the same order to the list of outputs
        3. Describe those new inputs/outputs and ammend the existing if it makes sense
4. Normalize the call name to use the **underscored** version
5. Add an Alias info to display the other name
6. Update the diary below accordingly
