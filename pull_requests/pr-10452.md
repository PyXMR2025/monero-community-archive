# Original Description
Turn the Monero fork into a standalone ASIC-resistant, CPU-mineable coin called Battleground, intended for use as in-game currency. RandomX (the proof-of-work Monero already uses, which strongly favors general-purpose CPUs and degrades GPU mining) is retained unchanged; all network-identity values are changed so a Battleground node cannot accidentally join the Monero network or call out to Monero infrastructure.

Coin-identity changes in src/cryptonote_config.h:
- CRYPTONOTE_NAME: bitmonero -> battleground (separates data directories)
- HASH_KEY_MESSAGE_SIGNING: Monero -> Battleground (wallet signatures not interchangeable with Monero's)
- NETWORK_ID: fresh UUIDs per nettype, so a Battleground node rejects Monero handshakes and vice versa
- Address prefixes (mainnet 66/67/68, testnet 96/97/108, stagenet 120/121/132) are disjoint from Monero's so a Monero wallet will not parse a Battleground address and vice versa
- Default ports (mainnet 19090/19091/19092, testnet 29090/.../.../, stagenet 39090/.../...) avoid collision with a Monero daemon on the same host
- GENESIS_NONCE bumped per nettype

Network isolation / stop phoning home to Monero:
- src/p2p/net_node.inl: empty get_ip_seed_nodes(); drop moneropulse blocklist DNS list
- src/p2p/net_node.h: empty m_seed_nodes_list (was seeds.moneroseeds.*)
- src/checkpoints/checkpoints.cpp: empty checkpoint DNS lists for all three nettypes (were checkpoints/testpoints/stagenetpoints.moneropulse.*)
- src/common/updates.cpp: empty update DNS list, placeholder download URL (was updates.moneropulse.* / downloads.getmonero.org)

User-visible branding:
- get_unit(): "monero/millinero/.../piconero" -> "battleground/milliground/ .../picoground"
- simple_wallet::set_unit(): accept new unit names
- `donate` command: message refers to the Battleground Project; donation address left empty until a canonical one is configured
- README.md rewritten with Battleground branding, RandomX rationale, coin parameters, bootstrap and in-game integration notes
