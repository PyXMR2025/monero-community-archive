# Original Description
Discussion: https://forum.monero.space/d/151-what-to-name-the-next-monero-release-018

Based on [Monero's release naming convention](https://github.com/monero-project/meta/blob/master/NAMING.md), Monero's next release name will be:

`Fluorine` + Celestial Object

"Alliteration or assonance is also required, for maximum awesomeness."

What should Monero's 0.18 release be named? Fluorine what?

Possible choices include (some of which stretch the definition of Celestial Object):

* Fluorine Fornax
* Fluorine Fath
* Fluorine Firewheel
* Fluorine Firmament
* Fluorine Furnace

...or anything else you come up with!

I'll run a ranked choice poll later as I have done the last few releases following initial feedback.
