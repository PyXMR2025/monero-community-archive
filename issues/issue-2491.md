# Original Description
On some computers, when I run `xmrig.exe` as administrator, I am told that Huge Pages will be enabled after a reboot.

So I reboot the computer. And on some computers, without running as Administrator, when I run `xmrig.exe`, it enables huge pages.

However on other computers after a reboot, if I run without admin rights, huge pages is unavailable but if I run it as admin then huge pages is enabled.

How come on some computers huge pages is enabled without running as admin rights but on other computers I do need to run `xmrig` as admin in order to enable huge pages?

I am using the exact same config files so it makes no sense to me?
