# Original Description
Easylogging is currently broken on platforms that do support the non-posix backtrace methods, but require linking to a specific library. This PR explicitly links to the right library.

The FindBacktrace.cmake module is included in cmake 3.0+, but since we also support cmake 2.8.7, I have copied this file - with a small modification - in the cmake folder.
