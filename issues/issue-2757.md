# Original Description
It would be nice to display the name of an address book contact besides its address in pages other than the Address book, like Transfer page (address field, transaction confirmation pop up) and Transactions page.

In order to do that, it is necessary a function like `getDescription(address)`, which accepts an address, checks if it is included in Address Book, and then returns the address name (description).
