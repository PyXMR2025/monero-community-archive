# Original Description
This adds a gitian build script and gitian descriptor to achieve reproducible builds in monero. 
Currently the build script in gitian-build.py still clones bitcoin's repository used to exchange, compare and store signatures. These two repos should be created on monero, namely gitian.sigs and ~monero-detached-sigs~. Edit: since no apple or windows certs are required, monero-detached-sigs is not needed. 
It also just builds Linux binaries at the moment. Gitian descriptors for OSX and windows still need to be created. 
To test this out, create a remote copy of this branch on your fork, tag a version to it in the format 'vN.N.N' and prepare the host as depicted in the README included in this pull request. Then setup and run a detached signature build.
Please post reported checksums below, so we see if the current setup already is reproducible, or if more tweaking is needed. I will post checksums as soon as my build is done.
