# Original Description
This PR adds fish shell completions for `monerod` and `monero-wallet-cli`. I am starting to use those binary files, and using `--help` all the time gets boring and slow. Maybe others that use Fish will find this useful too.

Basically, this is the output of `--help` adapted for Fish completions. I did a simple script for it and then manually imploved some things, such as adding `-a` flags, etc.
