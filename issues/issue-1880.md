## carrington1859 | 2021-11-04T09:38:08+00:00
It is also worth noting that there has been a Cambrian explosion of crowdfunding systems/ideas in recent months. Here are the examples I'm aware of.

Plowsof's wishlist-as-a-service (https://github.com/plowsof/xmr-wishlist-aaS)

This tool has been used by the following projects to set up independent crowdfunding

Plowsof's personal wishlist (https://plowsof.github.io/)
Monerujo (https://funding.monerujo.app/)
XMR Community art fund (https://moneroart.neocities.org/)
XMR.radio (https://xmr.radio/funding)

Plowsof also created a directory of various devs/projects with donation/contact info (https://plowsof.github.io/monerodevs/) & a BCH Flipstarter proposal to make the wishlist system multi-coin including stablecoins (https://wishlist.monerodevs.org/).

Outside Plowsof's efforts there are also the following:

Haveno bounties (https://github.com/orgs/haveno-dex/projects/2)
MAGIC Grants (https://www.magicgrants.org/) *Site seems broken at this moment*

I'm not suggesting that all of these ideas should get equal booking with the CCS, but it is worth having a discussion of if/how getmonero.org should promote these independent crowdfunding efforts.

Ultimately I think something like the monerodevs directory should be the aim. It seems to modeled on https://bitcoindevlist.com/ and the PoC was created in response to a bounty (https://bounties.monero.social/posts/7/setup-monero-dev-list-website). The bounty does specify "maintain a repo", but I'm not sure yet how actively maintained the project will be going forward.
