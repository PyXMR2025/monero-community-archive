## Spudz76 | 2019-10-29T21:59:15+00:00
Monero never "allowed" GPU mining, in fact every hardfork has been to destroy GPU and ASIC efficiency to keep it CPU only.  RandomX basically puts CPU back into the best H/W power band and injures GPU and ASIC to the point they are useless (unless you like running at a loss).

You will be unable to run with less than 4GB as well, RandomX requires 2080MB baseline locked memory (not just the scratchpad space like other CN algos).
