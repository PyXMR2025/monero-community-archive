# Original Description
Some preprints on the [MRL](https://web.getmonero.org/resources/research-lab/) page are hosted on the site, while others link to the [IACR preprint archive](https://eprint.iacr.org/), which is a generally trusted source for hosted cryptography preprints. I recommend the links to preprints be updated in two ways:
1. For external links, identify the destination: `Read paper (on IACR)`
2. For all preprints, link to the corresponding GitHub page for the TeX source: `View source (on GitHub)`
