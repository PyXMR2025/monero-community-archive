# Original Description
Together with the code for the GUI wallet itself from PR #2029 this implements custom URI schemes on Windows: The installer will register "monero:" and "moneroseed:" with Windows by writing corresponding registry entries.

As an advantage of letting the installer do this the two URI schemes will be deleted when uninstalling.
