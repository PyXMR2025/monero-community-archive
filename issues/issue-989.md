# Original Description
https://web.getmonero.org/resources/user-guides/howto_fix_stuck_funds.html

Right now the guide simply suggests the user to restore the wallet from seed, but we already have a guide for that purpose.

Would be better to rewrite the guide, adding suggestions on what to check and the steps to follow in case funds appear stuck. We should suggest to restore from seed only if no other alternative worked. In that case there is no need to write instructions about how to actually restore the wallet, we can just point to guides we already have: [How to restore your account](https://web.getmonero.org/resources/user-guides/restore_account.html) and [Restoring wallet from keys](https://web.getmonero.org/resources/user-guides/restore_from_keys.html).
