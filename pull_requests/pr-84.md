# Original Description
As suggested by Diego, and as I've noticed some time ago GPLv3 cannot be used anymore, so I've decided to migrate to a license that will allow proprietary software to use this library (legally).

Requirements comes down to
- Publishing modifications to source code of library
- Disclosing the license

according to https://github.com/MrCyjaneK/monero_c/graphs/contributors @sneurlax @Im-Beast @konstantinullrich have contributed to the codebase, so I'd like to publicly ask for permission to relicense the entirety of the project under LGPL. So please leave a commend that will say whether or not you agree to the license change.
