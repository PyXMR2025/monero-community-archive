# Original Description
I updated the Windows installer to Lithium Luna.

Other than for earlier updates I did not yet have the final, official 0.12.0.0 Monero GUI Wallet package from getmonero.org with the "authoritative" file list available, so there may still be one or the other error in the files listed in the InnoSetup script to install.

Some of the files and directories also depend on the exact version of Qt used to build and package Monero. I used whatever version the MSYS2 package `mingw-w64-x86_64-qt-creator` found on a Qt website [here](https://wiki.qt.io/MinGW-64-bit) currently installs.

But any needed corrections to this installer will be probably small.
