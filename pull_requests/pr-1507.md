# Original Description
As discussed in [Issue #2 of onion-monero-blockchain-explorer](https://github.com/moneroexamples/onion-monero-blockchain-explorer/issues/19), currently the serialized files of unsigned/signed transactions are unportable; e.g., [the tx pusher hosted by MoneroWorld.com](http://explore.moneroworld.com/rawtx) can work accept files created in Linux. This PR solves the problem.
