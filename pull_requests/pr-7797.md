# Original Description
I've discovered why ccache wasn't working under FreeBSD: because the cross compiler command was not an executable file, but rather a script, pointing to the exe file with all the options, that it needs. CCache expects an executable file. The solution was to parse and segment the script. I have described it in a comment of the changed file.
