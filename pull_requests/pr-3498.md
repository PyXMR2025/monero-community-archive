# Original Description
Transfer page has 2 buttons used to import:
- Scan QR code: can import address + amount request (QR code with amount request) or address only (QR code without amount request)
- Address book: import address from address book

Main Changes/Logic:
- Respect address and/or amount inserted by the user before importing
- If an amount was inserted by the user before importing a QR code with amount request, we will import QR code address + amount request in a new line. This is made so that the user can't be tricked into accepting a QR code requesting an amount higher than the amount previously set by the user.
- If last line address is blank and we are importing address, import it in the same line
- Delete "(all)" amount from the first line if the transaction has multiple recipients
- Allow adding new recipients if "infinite" button was clicked.
