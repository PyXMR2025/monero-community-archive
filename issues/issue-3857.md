# Original Description
![image](https://user-images.githubusercontent.com/21298211/157893347-27e69014-edd2-4e87-a27a-8873e4d2c6e8.png)
This Wallet tab under Settings should allow the user to lock the wallet. This is my workflow:
I log into my computer, the Monero-GUI is automatically opened (I have a shortcut in the Startup folder), and then I log in to start sync. I then create a new desktop to work. The first desktop has the GUI unlocked since there's no easy way to lock it. I'd have to set the timeout to a really low number and there are cases where a low timeout hinders UX. Therefore, a way to lock the wallet on demand would be really nice.
