# Original Description
monero-wallet-rpc has been extended to support watch-only wallets consistently with monero-wallet-cli.

wallet-rpc did not support watch-only wallets, mainly generating unsigned transaction set and submitting signed transaction sets. I've added this support to the RPC with minimal changes following the same approach used when implementing multisig (save, submit) to wallet-rpc.

Please, take a look if you are happy with the code. Merging this PR would be gold as I need watch-only wallets over RPC :)

Thanks!
