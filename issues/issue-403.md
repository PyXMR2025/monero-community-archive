# Original Description
Hey everyone, every hard fork we need to come up with a name for our release. The naming convention, as described [here](https://github.com/monero-project/meta/blob/master/NAMING.md) is a chemical element (in sequence) followed by a celestial object.

Last time was the Boron Butterfly release, and so now we have reached Carbon!

What are your ideas? Carbon what?

Carbon [Cigar](https://www.nasa.gov/feature/goddard/2017/messier-82-the-cigar-galaxy/)?

Carbon Cassiopeia? 

Let's hear it!

EDIT: Due to confusion below, I will clarify something that is in the Naming Convention doc. The celestial object should be an alliteration of the element. So since we are on Carbon, the celestial object should also start with a C.
