# Original Description
Changes:
- Change column order: Date + Address/Description + Amount. Most cryptocurrency wallets display transaction history in this order.
- Removed green and orange dots of the left side. 
- Vertically aligned "Pending" and "Failed" icons of the left side. Added tooltips to these icons.
- Removed "Sent" and "Received" text (this is indicated by orange "-" and green "+" on amount)
- Removed "To:" and "From:" on address field
- Moved Description below address/recipient. This feature is important in Monero and should be always visible.
- Moved blockheight to the right of Confirmations
- ~~Display fiat conversion (USD/EUR) below XMR amount (when enabled)~~ (will be included in a future PR)
