# Original Description
requires https://github.com/monero-project/monero/pull/1541
Still working on the UX, but the main functions should be in place. 

1. Create a view only wallet (settings page)
2. Open view wallet and create a transfer, save with "Save tx" button
3. In original wallet, click "Sign tx" on send page and select the file
4. in view wallet, click "Submit tx" and select the signed tx file

Preferably the main wallet should be installed on an air gapped computer not connected to the internet. The tx files can be transfered on a usb stick or other removable device.
