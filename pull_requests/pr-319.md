# Original Description
Uses the latest `monero-oxide` FCMP++ lib with the following changes:
   - [GBP Fix](https://github.com/monero-oxide/monero-oxide/commit/cba7117d2cb4a45444c54005604b2a943a8517f1)
   - [De-compress points on verify, not de-serialization](https://github.com/monero-oxide/monero-oxide/commit/756851892eb2d5a82b01e43f63b268127143ad62)
   - [Fast proof size calculation](https://github.com/monero-oxide/monero-oxide/commit/e11bae51bc182015a3e83fbb5ad2860a3245962d)
   
On the C++ side:
- Includes the optimizations to the `batchVerifyFcmpPpProofs` function from #224.
- Includes FCMP++ proof and tx benchmark tests.
