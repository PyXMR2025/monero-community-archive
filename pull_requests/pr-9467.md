# Discussion History
## hinto-janai | 2025-01-18T01:01:55+00:00
The post notes glibc 2.39, is this patch still needed?

## tobtoht | 2025-01-18T02:56:18+00:00
Yes. We build a custom cross-compilation toolchain that uses [glibc 2.27](https://github.com/monero-project/monero/blob/master/contrib/guix/manifest.scm#L181) instead of 2.39. This patch fixes that build.

We use a lower version of glibc to build Monero than what other Guix packages are linked against. This is because the glibc version we link against determines the minimum glibc version that must be available at _runtime_. A binary linked against glibc 2.39 will run on Ubuntu 24.04 (glibc 2.39), but isn't guaranteed to run on Ubuntu 22.04 (glibc 2.35). Ubuntu 18.04 is the lowest version we can support by using glibc 2.27.
