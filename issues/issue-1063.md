# Original Description
Please add a sha256 checksum to all downloads including media files. Currently there are no sha256sums for .pdf and .epub files.  
  
For any real security the checksums would need to be uploaded to other places as well. For example the monero subreddit and online chat rooms. Members from the community can proactively post checksums of media files. This is similar to how the checksums of the binaries are posted in various places.  
  
Please consider to cryptographically sign all files as well for extra security.  
  
Motivation: same reasons as with adding checksums and signatures to the binaries. Verify that you have the correct file. It can also help detect malicious changes to files.
