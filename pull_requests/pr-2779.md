# Original Description
Closes #2467

**To do:**
- Test Ledger integration
- Test sweep unmixable feature
- Test OpenAlias
- #753

**Changes:**

New TxConfirmationDialog component:
- From field: wallet name ("Ledger", "Trezor" or "My wallet") + account # + account label
- To field: displays icon + address book name (if address is in address book);
- Added fiat conversion to amount and fee
- Removed extra zeros from the end of amount and fee
- Keyboard can be used to cancel (ESC key) or to confirm dialogs (enter/return keys)
- When the dialog is waiting for user action, a blinking text message will appear (see Trezor example below)

Modified StandardButton component:
- Added secondary button: a grey button for not desired actions

Modified PasswordDialog component:
- Ok button text changes to "Send transaction" when sending tx
- Ok button text changes to "Save transaction file" when saving tx file in View only wallet

New SuccessfulTxDialog component:
- Added green circle checkmark animation
- View progress button: opens History page with transaction ID filled in search input
- Open folder button (view-only wallet): opens the folder where the transaction file was saved

History page:
- Destination address of pending (0 confirmation) transactions is now being displayed (fixes #2008)
- Search filter now works when there is a pending transaction (fixes #2703)
- Auto collapse when there is a single result
