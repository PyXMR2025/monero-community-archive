## jonathancross | 2017-05-04T17:42:26+00:00
Should double quote chars (`"`) also be stripped out?
I took a look at [what is allowed in git tags](https://git-scm.com/docs/git-check-ref-format) and did not see anything preventing quotes. Even if it is a very unlikely situation, it could cause problems when the `version.js` is created above.

Example:
```
% TAGNAME=$(echo 'foo"bar')
% echo "var GUI_MONERO_VERSION = \"$TAGNAME\""
var GUI_MONERO_VERSION = "foo"bar"
```
