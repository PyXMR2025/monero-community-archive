# Original Description
Specializing a trait to lie about the properties of a class is probably a bad idea. Instead, the requirements for hex conversion were relaxed to `is_standard_layout`. Microsoft has a decent [writeup of the traits](https://msdn.microsoft.com/en-us/library/mt767760.aspx). Although, the writeup never mentions that `is_standard_layout` must be contiguous (i.e. no vtables).
