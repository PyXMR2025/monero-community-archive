# Original Description
Kovri is a C++ router for I2P that is under heavy development. The intent is to run Monero [nodes](https://getmonero.org/knowledge-base/moneropedia/node) in the Kovri I2P router to hide IP addresses.

Kovri development helps Monero by increasing the privacy of it, and Monero helps Kovri by increasing the size of the I2P network.

View the Kovri project on GitHub [here](https://github.com/monero-project/kovri).
