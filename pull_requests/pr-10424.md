Yikes maybe the C version was better. The pure C++ approach:
```c++
using clock = std::chrono::system_clock;
const clock::time_point cutoff{std::chrono::seconds{req.recent_cutoff}};
if (m_restricted && clock::now() - cutoff > restricted_histogram_cutoff)
```

The system clock is guaranteed to be 1970 epoch starting in C++20; time_t is still unspecified but almost always the same 1970 epoch.
