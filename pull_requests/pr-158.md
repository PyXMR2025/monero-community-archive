# Original Description
For simplicity, using netcat (nc) can be easier than nmap since it comes pre-installed on all major distributions, avoiding installing additional packages for open port testing.

```
proxychains -q nc -vz monerotoruzizulg5ttgat2emf4d6fbmiea25detrmmy7erypseyteyd.onion 80

```
