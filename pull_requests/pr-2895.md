# Original Description
Closes #2467

**Changes:**

New TxConfirmationDialog component:
- From field: 
  - Wallet name ("Ledger", "Trezor" or "My wallet")
  - Account # + Account label (if the wallet has more than one account)
- To field:
  - Address book icon + name (if address is in address book)
  - Destination address
- Added fiat conversion to amount and fee
- Removed extra zeros from the end of amount and fee
- Keyboard can be used to cancel (ESC key) or to confirm dialogs (enter/return keys)
- When the dialog is waiting for user action (hardware wallet button press), a blinking text message will appear

Normal transaction:
