# Original Description

```
"hashrate": {
  "total": [12930.96, null, null],
  "highest": 12968.11,
  "threads": [
      [671.19, %  
```

This was previously mentioned in #290 and #3363

        [743.79, null, null],
        [723.8, null, null],
        [669.71, null, null],
        [720.22, null, null],
        [652.88, null, null],
        [740.84, null, null],

        [744.73, null, null]
    ]
