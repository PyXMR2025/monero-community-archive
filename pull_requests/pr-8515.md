The purpose is to pad by `d-r`, i.e., divisor - remainder, so the length is divisible by `d`. If `r==0` we return before appending.

## jbakosi | 2022-08-22T14:23:30+00:00
Since encrypt returns binary, we encode to [z85](https://rfc.zeromq.org/spec/32/). This is to please HTTP/JSON and printability. Other advantages are behind the link. Z85 is more efficient on the wire than hex or Base64.
