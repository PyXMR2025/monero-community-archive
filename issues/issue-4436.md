# Original Description
Currently the [Monero GUI on Flathub](https://flathub.org/apps/org.getmonero.Monero) has the status Unverified. Unfortunately, cryptocurrencies are often associated with scams, and therefore it would be highly beneficial for users' trust if the Monero team could complete the Flathub verification process. You can find more information about the verification process here: https://docs.flathub.org/docs/for-app-authors/verification

Thank you for considering it!
