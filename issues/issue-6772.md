# Original Description
Imagine you're a dilligent Monero user who regularly makes transactions to more than a dozen destinations.

Naturally, because address reuse in Monero is not a problem, you'll save the destination address and names/identifiers to the address book.

Now imagine you run a wallet on your laptop, desktop, and two mobile devices. And you want to share addresses between them.

It would be helpful to have a way to selectively (some entries) or entirely (the whole thing) export and import addressbook data between different wallets - presumably, they're all backed by monero-wallet-cli and the import/export would be compatible.

For only a few addresses you don't really notice the difference - I have been copying them manually forever, after all.

But when the address book grows beyond about two dozen ... this could really come in handy.
