# Original Description
Based on the contents of the official download file `monero-gui-win-x64-v0.12.3.0.zip` as released earlier today.

Mostly trivial adjustments to new release number, time passed, and some changes in the files to install.

With the confusion about the application name ("gui" versus "core") caused by 0.12.2.0 behind us, the installer can safely write any special blockchain location to the `monero-core` registry key where the GUI wallet will pick it up.

As an improvement, in the case of an *update* where the user could have modified the blockchain location in the GUI wallet already installed, the installer now picks up that location from the registry (instead of re-using stale data i.e. what the user said in the last installer run).
