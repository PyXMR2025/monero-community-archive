# Original Description
If we'd make a rct tx with just one input, we try to add
a second one to match the 2/2 ideal. This means more txes
use that template (and are thus using a larger anonymity
set), and it coalesces outputs "for free". We use dust
outputs in priority for this.

Further improvements could be made, like using the smallest
(or one of the smallest) amounts, but this would need some
thinking to see what is best there.
