# Original Description
PR #3478 (v0.12.0) rebased on master after merge

Add the possibility to export private view key for fast scan.
On client startup the device asks for authorization to export the private view key. If user
agree, the client hold the private view key allowing a fast blockchain scan. I user do not
agree, the blockchain scan is fully done via the device.

When creating/restoring wallet, if --restore-height option is not used the current estimate
height is used for starting the scan. In other words it is assume we are creating a new account.
