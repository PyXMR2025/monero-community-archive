- https://github.com/monero-project/monero/pull/9367

Open question is whether we want to replace this with PCP / NAT-PMP first. Bitcoin Core uses a bespoke [implementation](https://github.com/bitcoin/bitcoin/pull/30043) (disabled by default) and previously used [libnatpmp](https://github.com/miniupnp/libnatpmp), another C library maintained by the miniupnpc dev.

Passing `--no-igd` to `monerod` will now result in an error. We may want to make it a no-op instead.

Opened for feedback.
