# Version: v0.15.0.4

# Release Notes
# Overview

This is the v0.15.0.4 release of the Monero GUI software.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.15.0.1)

Some highlights of this release are:

- Various small UI improvements
- New langauge: Norwegian
- macOS: .app now displays version
- Bug fixes and performance improvements
- Crash fixes
