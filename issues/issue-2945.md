# Original Description
Steps to reproduce:

1. Run monero-gui-mac-x64-v0.16.0.0
2. Restore a mainnet wallet from seed, with no restore height specified. Use a random seed generated by https://moneroaddress.org/. Use a remote node, such as node.supportxmr.com
3. Observe that the balance will show as ?.?? for several hundred thousand blocks, and then suddenly switch to display 0.000000000000000 even though there are hundreds of thousands of blocks left to scan.

I'm concerned that people may misinterpret this as the wallet suddenly completing the sync and reporting an empty balance.
