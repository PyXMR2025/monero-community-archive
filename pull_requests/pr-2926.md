# Original Description
(The idea was born in a Reddit thread: https://www.reddit.com/r/Monero/comments/7jll1b/multisig_safety_subaddresses_mental_model/dr8cprh/)

This patch introduces the idea of "tagging" accounts in order to group arbitrary subsets of accounts for easier finance management. The following new sub-commands are added to the `account` command:

    account tag <tag_name_1> [<tag_name_2> ...]
    account list_tags
    account add_tag <tag_name>
    account delete_tag <tag_name>

Example usage is below:
