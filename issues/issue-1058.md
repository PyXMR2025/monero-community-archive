# Original Description
The [guide of the GUI wallet](https://github.com/monero-ecosystem/monero-GUI-guide/blob/master/monero-GUI-guide.md) could be added to the 'Wallets' section of the 'User Guides' page: https://web.getmonero.org/resources/user-guides

Instead of adding the guide itself, i think would be better to add a link to the guide on GitHub. We could list it with the other guides, but we would warn the users about the fact that they are clicking on an external link. This could be also useful in case we decide to add more external resources to that page.

As suggested on reddit: https://www.reddit.com/r/Monero/comments/hg60sb/new_release_of_the_guide_for_the_gui_wallet/fw1y9ur/
