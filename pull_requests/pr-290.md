Closes: https://github.com/Cuprate/cuprate/issues/282

This pull request split the `BlockBlobs` database table into two new tables: `BlockHeaderBlobs` and `BlockTxsHashes`.
```rust
/// Serialized block header blobs (bytes).
///
/// Contains the serialized version of all blocks headers.
0 => BlockHeaderBlobs,
BlockHeight => BlockHeaderBlob,

/// Block transactions hashes
/// 
/// Contains all the transaction hashes of all blocks.
1 => BlockTxsHashes,
BlockHeight => BlockTxHashes,
```

<!-- Describe the pull request in detail. -->

#### Subsequent modifications have been made:
- VerifiedBlockInformation now have a `mining_tx_index: u64` field that contains the canonical identifier of the mining transaction of that block in the database.
- `add_block`, `pop_block` and `get_block_extended_header` have been edited consequently.
