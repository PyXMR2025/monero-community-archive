# Original Description
This micro change made it possible to reduce the visibility of the logging class and the `<algorithm>` header down to the implementation file.

The PR is intentionally small, not to abuse the reviewers' time. But as such, there will most probably be no noticeable gains from this PR alone.

I kept the trivial calls as inline, while all longer calls are moved to the corresponding .cpp file.
