## selsta | 2021-12-17T15:05:05+00:00
We don't rely on 3rd party packages. The official place to download monero / monero-gui is here: https://www.getmonero.org/downloads/

Anyone can decide themselves if they
- download the official binaries
- compile the program themselves
- use their package manager (which are mostly community maintained)

We don't have the resources to "officially" maintain packages for every package manager out there, so the only official place to download monero is the website.
