# Original Description
Recompress all PNG images using [zopfli](https://github.com/google/zopfli). Slightly smaller than #1095 and covers all PNGs.

The downside to doing all images is that the old images are still floating around in the Git history, adding a little to the initial clone time for developers.

##### Commands to reproduce

```bash
mkdir img-old img-new
cp img/*.png img-old/
for pic in img-orig/*.png; do
    echo zopflipng img-orig/$(basename $pic) img-new/$(basename $pic);
done | parallel
du -h img/*.png img-new/*.png
```
