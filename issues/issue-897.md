Google Images and other search engines offer the ability to refine searches to only include images that meet certain license requirements. Since the getmonero.org website images and content are available to all, we should explicitly tag all the images as such so that search engines display these images along with other results.

Google recommends doing this with RDFa: https://webmasters.googleblog.com/2009/08/specifying-images-license-using-rdfa.html
