# Original Description
@j-berman's idea to filter out dust automatically: https://github.com/seraphis-migration/monero/pull/51#issuecomment-2931594820

This implementation removes dust by trying to include dusty inputs in a transaction of the most optimal input count if a dust output doesn't pay for itself at other input counts. If there is no input count for which the output pays for it's own marginal fee, then it is ignored.
