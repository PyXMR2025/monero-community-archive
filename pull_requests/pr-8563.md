# Original Description
Ubuntu 22.04 build target `release-static` requires the addition of the libbsd and libmd libraries for libzmq, otherwise the build fails.

It also requires you remove the libzmq3-dev installed from the main ubuntu repository and instead compile it yourself:

```
git clone https://github.com/zeromq/libzmq.git
cd libzmq
./configure --enable-static --disable-shared --disable-libunwind --disable-perf --with-pic --without-docs --with-libsodium
make
sudo make install
cd ../
```
