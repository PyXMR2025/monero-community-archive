# Original Description
**Change summary**:

**1) Wallet mode selection page** 
- At first, only two options appears: simple and advanced mode
- Simple mode icon is simpler than the advanced mode icon
- ~~Added _(recommended)_ tag to simple mode~~ (see comments below)
- Description of simple mode includes only features, and doesn't explain connection options
- Included transactions history, address book and hardware wallet support in simple mode description. We could also add merchant mode and accounts, but decided not to include them.
- Advanced mode description now includes almost all advanced features available
- Removed the "_The blockchain is downloaded to your computer_" from advanced mode description, since users can use advanced mode with remote node (without downloading the blockchain).

![image](https://user-images.githubusercontent.com/45968869/82127520-7caf6b80-97b4-11ea-97d4-2aeca613b8df.png)

**2) After selecting simple mode, two connection options for simple mode are displayed (actually each option is a wallet mode, but I decided to not call them modes)**
- ~~Added _(recommended)_ tag to Local Node + Bootstrap node option (wallet mode 1)~~ (see comments below)
- Moved wallet mode 1 to the first option, since it's the recommended one
- Added FontAwesome icons (the same used in Settings > Node page)
- In both options added "remote node" term in the description and remote note cloud icon, to make it clear that simple mode always connects to a remote node (temporarily or permanently).

![image](https://user-images.githubusercontent.com/45968869/82127528-92bd2c00-97b4-11ea-914e-b326f2eed4a2.png)

**3) Warning after selecting Local node & Bootstrap node**
- Better explanation overall
- Precise instructions for users who prioritize privacy: they should use an advanced mode and start a local node without setting a bootstrap node (previously it was only said that those users should use a "full node")

![image](https://user-images.githubusercontent.com/45968869/82120834-27119980-9789-11ea-80fa-4d7156b4d74a.png)

**4) Warning after selecting Remote node**
- Better explanation overall
- Precise instructions for users who prioritize privacy: they should use an advanced mode and start a local node (previously it was only said that those users should use a "full node")
![image](https://user-images.githubusercontent.com/45968869/82120855-3690e280-9789-11ea-917e-1744f07b7c9d.png)
