# Original Description
The goal of this PR is to best differentiate the expressions "Create new wallet" (create a wallet file with zero balance) vs. "Restore a wallet" (create a wallet file from an old wallet with some balance).

When restoring a wallet (from keys/seed or hardware wallet):
- Wizard summary subtitle will display "Details of the wallet to be restored:" instead of "New wallet details".
- Wizard summary will display "Restore wallet" button instead of "Create wallet" button.
