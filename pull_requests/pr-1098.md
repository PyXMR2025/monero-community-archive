# Original Description
#### _config.yml: Change baseurl to https://www.getmonero.org

baseurl was set to https://getmonero.org. This caused a redirect from getmonero.org to web.getmonero.org every time the user clicked on a link served using site.baseurl (basically all internal links). A lot of time was wasted in the redirect (between 500-900ms or even >1000ms with my slow connection). This will make switching from page to page much faster and will improve the SEO.

The problem was related to a misconfiguration in the CDN. @fluffypony now set the canonical base url to `https://www.getmonero.org`, so we are changing to that `baseurl` and `url` to fix the problem described above (the initial PR changed the baseurl to `https://web.getmonero.org`).

#### Shrink heavy images in 'Accepting' and 'What is Monero' 
account_switch.png from 35KB to 8KB
merchant_page.png from 107KB to 30KB
Monerujo-wallet.png from 252KB to 101KB
receive.png from 92KB to 26KB
ringct-poster.png from 120KB to 44KB
ring-signatures-poster from  75KB to 47KB
stealth-address-poster.png from 108KB to 84KB

This PR should be merged along with #1097 to avoid the creation of two different sitemap files.
