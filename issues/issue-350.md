# Original Description
I believe that the Monero community would benefit significantly from a self-hosted Jitsi solution. Jitsi is an open-source videoconference tool. We use it for the Coffee Chats, Breaking Monero series, and more.

Jitsi graciously provides a free service at https://meet.jit.si, but it lacks a few key features:

- [ ] It only supports up to 720p. Ideally, it should support at least 1080p.
- [ ] It includes the Jitsi logo in the upper left. Ideally, it would have a small Monero logo or no logo.
- [ ] The livestream feature is unreliable.

I believe that we have used Jitsi enough in the past to warrant a self-hosted instance of Jitsi if we can manage.
