# Original Description
With use-it-or-lose-it power like solar and hydro, it would be nice to have an option to switch over to donating when an active pool is not available (e.g. p2pool node downtime, maintenance, etc.)

Even nicer would be to do some simple accounting to credit the upcoming donation levels, but the main idea is to not waste available free power when it could help the networks.

Should be off by default since most people pay for power but if an idle miner can benefit the developers at no cost, why not?
