# Original Description
Add two new methods `generate_from_keys` and `generate_from_view_key`. These methods are useful if the key generation is outside of the wallet, e.g. in an other library.

Example of `generate_from_keys`:

```
curl -X POST http://localhost:18083/json_rpc -d '{"jsonrpc":"2.0","id":"0","method":"generate_from_keys","params":{"restore_height": 0, "filename": "test", "address": "48tgL9CicAra88CfK5giqYRxbXTLEBvva1XCpc7GwNTGLxX4uGXE8neK9oejJ8dJUKGStW2Mu3PfrMhr3u6gymP47xfV9YE", "spendkey": "183818b6222c89dec1216810aa56c3c1262271b04af1cadd030c93a4bcf75908", "viewkey": "183818b6222c89dec1216810aa56c3c1262271b04af1cadd030c93a4bcf75908"}}' -H 'Content-Type: application/json'

{
  "id": "0",
  "jsonrpc": "2.0",
  "result": {
    "address": "48tgL9CicAra88CfK5giqYRxbXTLEBvva1XCpc7GwNTGLxX4uGXE8neK9oejJ8dJUKGStW2Mu3PfrMhr3u6gymP47xfV9YE",
    "info": "Wallet has been restored successfully."
  }
}
```

Example of watch-only wallet with `generate_from_view_key`:

```
curl -X POST http://localhost:18083/json_rpc -d '{"jsonrpc":"2.0","id":"0","method":"generate_from_view_key","params":{"restore_height": 0, "filename": "test", "address": "48tgL9CicAra88CfK5giqYRxbXTLEBvva1XCpc7GwNTGLxX4uGXE8neK9oejJ8dJUKGStW2Mu3PfrMhr3u6gymP47xfV9YE", "viewkey": "183818b6222c89dec1216810aa56c3c1262271b04af1cadd030c93a4bcf75908"}}' -H 'Content-Type: application/json'

{
  "id": "0",
  "jsonrpc": "2.0",
  "result": {
    "address": "48tgL9CicAra88CfK5giqYRxbXTLEBvva1XCpc7GwNTGLxX4uGXE8neK9oejJ8dJUKGStW2Mu3PfrMhr3u6gymP47xfV9YE",
    "info": "Wallet has been restored successfully."
  }
}
```
