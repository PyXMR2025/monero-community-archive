# Original Description
I didn't see this suggested anywhere, so here goes.
When running on Windows in GUI mode, it would be nice if the application could be banished to the system tray.  This gets it out of the taskbar and makes it a little less intrusive. Out of sight, out of mind and all that.

Since the application is meant to run in the background for long periods of time, this makes sense to me.
