# Original Description
The Monero wallet file is not simple or advanced, it's always of the same type. What changes is the user interface mode (simple vs advanced) set in Monero GUI software.

Some simple mode users may think that they have created a wallet file that doesn't support advanced functions, and now they must create a new "advanced" wallet file.

Some simple mode users may strange when they open their wallet file in a Monero GUI set to advanced mode.
