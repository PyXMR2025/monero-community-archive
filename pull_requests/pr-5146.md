# Original Description
Building with docker is arguably easier and more familiar to most people
than either kvm, or lxc.

This commit also relaxes the back compat requirement a bit. 32 bit linux
now uses glibc version 2.0. Also, the docker shell could not handle gcc arguments
containing spaces, so the explicit '-DFELT_TYPE' declaration was dropped.

Remove some packages from the osx descriptor.

To quickly test this on linux, install docker on a debian-like system (needs apt-get) and create a fresh `gitianuser` user. Make sure the user has full permissions to docker, or run the build script with `sudo`. Then clone my fork of monero: `git clone git@github.com:TheCharlatan/monero`. Copy the `gitian-build.py` script to the gitianuser's home directory and run:

```bash
./gitian-build.py --setup --docker --url https://github.com/TheCharlatan/monero testuser 4a9257b46438b5fb8fe5d58ee8c3b2bbe2b826cc
./gitian-build.py --build --no-commit --detach-sign --docker --url https://github.com/TheCharlatan/monero testuser 4a9257b46438b5fb8fe5d58ee8c3b2bbe2b826cc
```
