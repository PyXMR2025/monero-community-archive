# Original Description
Currently `Send` page always open with a colapsed `Description` field.

We could add an option `Display Description field in Send page` in Settings > Interface, that would make the Send page always open with the Description field expanded.

This way users that frequently use the `Description` field will have to click one thing less every time they send a transaction. For instance, they could just hit Tab in the keyboard to move to the next field.
