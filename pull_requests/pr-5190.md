# Original Description
Command line switch `--public-node`
1. Requires restricted RPC mode
2. Advertise node's restricted RPC port to other peers over P2P network
3. Print a warning at startup if restricted RPC server is listening on local or loopback address
4. Store remote peer's RPC port in peer list only if the peer's destination address is reachable

The following commands will include remote peers' RPC ports (if any):
1. `print_pl` daemon command
2. `get_peer_list`, `get_connections`, `sync_info` RPC calls

```diff
monerod --public-node
- ERROR, restricted RPC mode is required

monerod --public-node --restricted-rpc
+ OK, default RPC port will be advertised over P2P network

monerod --public-node --rpc-restricted-bind-port 1234
+ OK, RPC port 1234 will be advertised over P2P network

monerod --public-node --restricted-rpc --rpc-restricted-bind-port 1234
+ OK, RPC port 1234 will be advertised over P2P network
```
