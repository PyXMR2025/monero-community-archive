# Original Description
## 1
1. OpenCL RandomX synchronization reduction
2. CPU hot path algorithm dispatch cache
3. Shrinking the critical section of `Miner::setJob()`
4. Deleting redundant input uploads for OpenCL KawPow
## 2.1 OpenCL RandomX synchronization reduction
Modified files:
- `src/backend/opencl/runners/OclRxBaseRunner.h`
- `src/backend/opencl/runners/OclRxBaseRunner.cpp`
- `src/backend/opencl/runners/OclRxJitRunner.cpp`
Changes:
- Removed the redundant `OclLib::finish(m_queue)` at the end of `OclRxBaseRunner::run()`.
- Changed RandomX dataset upload from `CL_TRUE` to `CL_FALSE`.
- Introduced a persistent staging buffer for the job blob:
  - `m_inputBlob`
- Changed job blob upload from blocking write to asynchronous queue write.
- Removed `OclLib::finish(m_queue)` in `OclRxJitRunner::execute()`.
- Reduced unnecessary blocking of the host thread caused by `clFinish` / blocking writes.
## 2.2 CPU Hot Path Algorithm Dispatch Cache
Modified files:
- `src/backend/cpu/CpuWorker.h`
- `src/backend/cpu/CpuWorker.cpp`
Changes:
- Cache during the `consumeJob()` stage:
  - `m_jobFamily`
  - `m_jobHashFn`
- In the hot loop, no longer repeatedly call:
  - `job.algorithm().family()`
- Move the "algorithm parsing/function lookup done on every hash" out of the hot loop.
- Low risk, and fully fits the current `CpuWorker` structure.

## 2.3 `Miner::setJob()` Critical Section Reduction
Modified files:
- `src/core/Miner.cpp`
Changes:
- Update `d_ptr->algorithm` inside the lock.
- Execute GhostRider's `initGhostRider()` main path outside of the global state lock.
- Improve concurrency neatness on the job-switching path.
- Does not change steady-state computation logic.
## 2.4 OpenCL KawPow Redundant Input Upload Removal
Modified files:
- `src/backend/opencl/runners/OclKawPowRunner.cpp`
Changes:
- Removed the blocking `enqueueWriteBuffer(m_input, CL_TRUE, ...)` in `set()` that would be immediately overwritten by the next `run()`.
- Reduced one unnecessary host-device synchronization in the KawPow job switching path.
