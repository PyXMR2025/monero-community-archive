# Original Description
I propose a name change to the project and all related uses of the word Monero to Moonero. This is an important and necessary branding change that will help ensure the success of the project. Let's be honest, we all know Monero is a weird name, and most people ignore it because it sounds Mexican. With a name like Moonero, the world can embrace this trailblazing project and give it the hype it needs to propel it to greatness.

I've done the hard work of replacing all the names in this README.md doc, the rest is up to you!
