# Original Description
Room: https://hub.link/ktcMqWR

While this is meetup like, given it's serving also as a test bed for virtual alternatives to physical meetups (e.g. Konferenco) figured it should have a meta issue. Organized by rehrar. Original post can be found [here on Reddit](https://www.reddit.com/r/Monero/comments/fhlie9/monero_vr_meetup_march_15_17_utc_vr_not_required/):

> The meetup:
> 
> https://hubs.mozilla.com (exact room to be announced). Browser based. No VR required. Feel free to test it out today. Come join us as we see what this platform is capable of.
> 
> The background:
> 
> Hey everyone. So as the Konferenco got cancelled and virus and all that, I've been looking into alternatives for meeting up, running conferences, and doing other fun stuff. I mentioned as much on the last coffee chat.
> 
> In that coffee chat I mentioned AltspaceVR, and while I think it's less clunky than the Mozilla one, it is no open source, and doesn't support Mac (with partial Linux support), whereas the Mozilla one is browser based and will work for everyone.
> 
> We're going to do a casual meetup. Say hi. Maybe have some people say what they've been working on. Come by just to see how many people we can fit in such a meeting before the server starts to chug (so we can see if this is a legit option for meetups or conference type stuff in the future).
> 
> See you all there!
