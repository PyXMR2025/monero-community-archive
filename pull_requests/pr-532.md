# Original Description
When throwing an exception from being unable to begin an LMDB transaction, include the reason.

It's often been due to a write transaction attempted within a write transaction (batch mode), but there can be other reasons such as write transaction attempted while database was opened read only, or
environment's map needs to be resized.
