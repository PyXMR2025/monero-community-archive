# Original Description
In Transactions page, the transaction key is displayed after the user clicks in "Click to reveal" link.

However, when the Transactions page auto-refreshes itself, the transaction key is hidden again and the user must click the "Click to reveal" link to see it again. 

It is a strange behavior because the page auto-refreshes without warnings. I clicked in "Click to reveal" link and in less than 2 seconds it was hidden again without warnings (page had autorefreshed).
