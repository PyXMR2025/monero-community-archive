# Original Description
When you select an account in Account page or an address in Receive page, the selected item is not highlighted enough, because only the number (`#1`, `#2`, `#3`, etc.) changes color.

I suggest we change the background color of the selected item to the same one used when you hover the mouse cursor over it.

Also, all text in the selected item should change color, not only the index number.
