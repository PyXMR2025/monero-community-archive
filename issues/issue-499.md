# Original Description
Sections are only outlined if they have multiple components. For example, in the Sign/verify tool they give a gray border line to the Sign section and to the Verify section.

![signverify](https://cloud.githubusercontent.com/assets/21302237/23380003/1c992bfa-fd07-11e6-9958-1b6d9cda10a6.png)

For the Check Payment tool there is only one component, so there is no gray outline.

![checkpayment](https://cloud.githubusercontent.com/assets/21302237/23380015/2a88728e-fd07-11e6-99ad-4be79f0cb507.png)

Currently, there's a gray outline around the Solo Miner because originally it was paired with a Pool Mining option. Since the Pool Miner was never merged, the gray border of the Solo miner should be removed, keeping it more in line with single component pages like Check Payment.

![solominer](https://cloud.githubusercontent.com/assets/21302237/23380025/32b7e930-fd07-11e6-8c18-9fd367469ca7.png)
