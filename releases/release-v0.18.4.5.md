# Version: v0.18.4.5

# Release Notes
# Overview

This is the v0.18.4.5 release of the Monero GUI software. This release fixes a bug with Ledger hardware wallet.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.18.4.5)

Some highlights of this release are:

- Fix Ledger Monero app crash (#4534)
- Add support for Ledger Nano Gen5 (#4539)
- Update P2Pool to v4.13 (#4543)
- Improve macOS dark mode support (#4537)
- Minor bug fixes
