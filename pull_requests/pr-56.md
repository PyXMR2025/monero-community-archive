# Original Description
Adds `crossbeam` to the workspace's dependencies, and replaces `helper/`'s atomic float implementation with `crossbeam::atomic::AtomicCell<float>`.

As noted in https://github.com/Cuprate/cuprate/pull/45#issuecomment-1898503471 this:
- Is lock-free
- Allows `const fn new()`

A static assertion is also added to make sure targets that don't support lock-free atomic `32/64-bit` operations fail.
