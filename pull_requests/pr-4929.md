# Original Description
To ensure that the binaries compiled by gitian run across many linux distributions, enforce 2.17 as the minimum libc version supported. This means that libc functions defined in versions above 2.17 either need to be re-defined, or have an alternative version selected form the libc library with a `__asm` instruction. 
This commit works in combination with https://github.com/monero-project/unbound/pull/12 .
