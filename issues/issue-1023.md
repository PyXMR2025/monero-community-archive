# Original Description
Since I don't know if this feature is actually useful to anyone else, here's an open issue to get input on the matter.

Presently, Coveralls will spam (and is spamming) GitHub notifications with commit comments on watched repos; this includes bitmonero. This makes it difficult to read actually useful human comments in PR's and commits.

Un-checking "Leave Comments?" in https://coveralls.io/github/monero-project/bitmonero/settings should solve the problem and won't effect coverage.
