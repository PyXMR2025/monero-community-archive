# Original Description
In serialization/binary_archive.h, the ios_base::streampos member type was used, which is deprecated (and removed in C++17). (see [cppreference.com](https://en.cppreference.com/w/cpp/io/ios_base)) The usage is changed to ios_base::pos_type, which is not deprecated.

In cryptonote_core/tx_pool.h, the function call overload of a comparison object was not marked 'const', which it should be for usage with STL containers. (I don't know why this compiled fine otherwise.)

Also, in cryptonote_core/blockchain.cpp, an enum value was used as a boolean condition. Since the enum value was nonzero, the condition was always true; I think the original code was a typo, and this fixes it.

Compiler: GCC 8.2.1
