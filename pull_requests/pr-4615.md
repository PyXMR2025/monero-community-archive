# Original Description
Changeset: `rescan_bc` command implemented in WalletAPI as `Wallet::rescanBlockchain()` and `Wallet::rescanBlockchainAsync()` methods.

Rationale: This functionality was required by mobile wallet team.

Notes: `WalletListener::refreshed()` callback is called only after `Wallet::rescanBlockchain()` is finished (It could be requested in background thread even while earlier `Wallet::refreshAsync()` is going on). This is implemented as a cycle in `doRefresh()` until `m_refreshShouldRescan==false`
