# Original Description
Closes #3818

When changing network type without changing the wallet path the gui will attempt to open the most recently used wallet path and network type even if they don't match. This pull request forces the user back to the wallet setup under this condition. As long as network type and wallet path are both updated before closing the issue will no longer occur.
