# Original Description
Suggestion: skip the popup to set the label of a new address when creating a new receive address. Use the same input structure as the "Send" page. Make the label/description input section for the new address part of the Receive page (but don't hide it [behind a + symbol](https://github.com/monero-project/monero-gui/issues/3068)).  
  
Use one term to describe the label/description on the "Send" and "Receive" page. The "Send" page mentions "description" and the "Receive" page mentions "label".  
  
Change the "Create new address" button to the same color and size as the "Send" button on the "Send" page.  
  
Always show "Create new address" button and QR code section without having to scroll up or down.  
  
This [PR](https://github.com/monero-project/monero-gui/pull/2936) by @rating89us already implements some of these ideas and looks pretty good.
