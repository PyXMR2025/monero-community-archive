# Original Description
So this is my attempt at an implementation of specifically the 2/2 multisig scheme. I chose this limitation to begin with because it requires fewer intermediary steps for wallet and transaction creation because there's less secret data to exchange.

This build depends on monero-project/monero#7357

The steps are pretty similar to that of the CLI wallet, just visual. It could probably use some UI/UX improvements later on including a cool icon on the first page of the wizard like there is for the hardware wallet option.
