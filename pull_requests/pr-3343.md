# Original Description
The terminology of "mixins" and "ring sizes" are often confused with each other. If I recall correctly, the term "mixin" is more or less being "phased out" in favour of just refering to the ring size. After seeing somebody get confused about this on, IRC I made this change to use the desired ring size as the param for the "transfer" command for the rpc wallet in place of the mixin to avoid confusion.
