# Original Description
OpenBSD unfortunately ships with an old version of readline in its base system.

This causes artifacts on the screen when using the wallet cli and random core dumps.

For now, the best fix is to turn readline off on OpenBSD until I can figure out how to get the newer version of readline working. It can be added to the system using pkg_add, but for some reason it doesn't link cleanly yet.

This fix has been confirmed to work on OpenBSD 6.9, OpenBSD 7.0, OpenBSD 7.1, and OpenBSD 7.2.

Edit: also includes slight changes to the OpenBSD instructions to keep in sync with the changes on OpenBSD that have happened over time.
