# Original Description
Windows x64 GUI Wallet 0.12.0

If you type in an any field on a wallet, then in-wallet close that wallet and open another, the text in that field is still typed in that field in the different wallet.

Does not appear to be a serious issue, however could raise concerns regarding privacy if multiple users share same desktop and use wallet app.
