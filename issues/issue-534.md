# Original Description
Some merchants in our list use external converters that usually work with bitcoin (like shapeshift, coinpayments, changelly etc..). That means an user must send money to a third part when making a transaction with the vendor. The resulting transaction didn't happen completely inside the Monero network, making the exchange of money between parts not completely private, since this "mediators" can log valuable informations.

The user should be aware of this when choosing a merchant, an [Uses Converter] label on the side of the merchant's name easily notify this and push vendors to adopt Monero directly.
