Simple proof for `get_block`:
```bash
curl \
    http://127.0.0.1:18081/json_rpc \
    -d '{"jsonrpc":"2.0","id":"0","method":"get_block","params":{"height":0}}' \
    -H 'Content-Type: application/json'
```
