## jroelofs | 2017-12-27T15:49:48+00:00
Clang doesn't support these, so to reduce warning noise in the builds:

```
if (CMAKE_CXX_COMPILER_ID MATCHES "GNU")
  set(LOWMEM_FLAGS "--param ggc-min-expand=10 --param ggc-min-heapsize=65536 -O1")
else()
  set(LOWMEM_FLAGS "-O1")
endif()
```

I would have suggested adding `-Werror=unused-command-line-argument` to the `check_*_compiler_flag`s and running these GC flags through `add_*_flag_if_supported`, but gcc doesn't support that warning :(

## jroelofs | 2017-12-27T15:51:52+00:00
