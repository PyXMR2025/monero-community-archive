# Original Description
Updates InProofV1, OutProofV1, and ReserveProofV1 to new V2 variants that include all public proof parameters in Schnorr challenges, along with hash function domain separators. Includes new randomized unit tests.

While the unit tests include checks for cross-version proof generation and verification, this has not been specifically checked in functional tests, since there is no exposed way to easily generate a V1 proof from a transaction anymore.
