# Original Description
Right now, HTTP headers and the HTTP body are sent separately. This PR change sends the bodies and headers combined when appropriate. For small bodies, this can reduce the number of packets sent by 1. For a series of HTTP requests made with small bodies, the number of round trips can be reduced by up to 50%. 

You can test this by analyzing HTTP traffic with a packet sniffer like Wireshark.
