# Original Description
This change enables adding support for new architectures without having to add CryptoNight support for those architectures.

This will help with #2227, since without these changes [the code to support RandomX on POWER / ppc64(le)](https://github.com/cyrozap/xmrig/tree/wip-power-support) requires stubs for CryptoNight, which is not ideal.
