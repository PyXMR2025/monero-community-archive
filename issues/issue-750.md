# Original Description
Monero GUI at the moment is **extremely slow**, the reason is the synchronizing.

I have spent 3 days trying to just set up and use a Monero wallet, this is a crazy shameful amount of time wasted and it makes me very concerned the coins i bought are a bad investment since others will also struggle during setup. This is a **major** barrier to Monero being used by ordinary people.

How do i connect the Monero GUI to a remote node?

There should be easy GUI options so i can just open Monero Wallet, skip the synchronization and select from a list of remote nodes. Then i could be using Monero in ~5 minutes instead of the currently shameful ~3 days required.

Please fix this, Monero is practically unusable until new users can skip this synchronization bull#$@!.
