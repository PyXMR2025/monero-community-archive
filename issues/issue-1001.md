# Original Description
At the moment we have a "Recent News" tab in the navigation menu and when we hover on it, we will see 5 submenus (Urgent, Releases,Community, Meeting logs, all posts).

I propose to replace the "Recent Nnews" menu and submenus in a single "Blog" tab, without submenus. This way people will just need to click on "Blog" and all posts will show up (same effect of clicking "All posts") then we can give people the possibility to filter all blog posts according to the categories we already have (Urgent, Releases,Community, Meeting logs, all posts), using something like the "polular tags" box, but containing the 5 present menu categories.

This will save some clicks to the visitors and will declog the navigation menu, which is already quite dense.
