# Original Description
Closes #3137, closes #2468

Main changes:
- Wizard: implement getApproximateDate() function, which uses a blockheight as input
- "Date from" minimum date = Approximate date of restore height
- "Date from" maximum date = "Date to" selected date
- "Date to" minimum date = "Date from" selected date
- "Date to" maximum date = Today
- Add "Restore height" (start of wallet synchronization) and "Today" (end of wallet synchronization) links
- Mouse wheel considers minimum and maximum date

Imported from other PRs
- Highlight selected date (orange box) (PR #3532)
- Highlight today (yellow text) (PR #3532)
- Invalid days (before minimum date and after maximum date) are hidden and can't be selected (PR #3532)
- Font Awesome fallback icons for left/right arrows (previous/next month) and down/up arrows (open/close calendar)  (PR #3531)
- Go to previous/next month with mouse wheel (a lot faster than clicking on left/right arrows) (PR #3533)
