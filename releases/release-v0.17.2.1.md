# Version: v0.17.2.1

# Release Notes
# Overview

This is the v0.17.2.1 minor point release of the Monero GUI software.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.17.2.0)

Some highlights of this minor release are:

- Fix transfer on macOS (#3390)
- Minor bug fixes
