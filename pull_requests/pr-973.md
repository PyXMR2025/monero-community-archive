# Original Description
Removed all _data/LANG/roadmap.yml files and moved all strings to _i18n/LANG.yml to take advantage of the multilingual plugin.

Also removed all loops in resources/roadmap/index.md, now we simply map to each placeholder. If some strings were not translated, we only transfer the empty placeholders so that the plugin knows it has to use the english versions.

This PR fixes the problem of the switched entries on weblate (For reference, see #957) that the first changes fixed only partially.

I tested this PR on my personal Weblate instance.
