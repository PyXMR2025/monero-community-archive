# Original Description
Changes:
- Move password strength meter below the password field
- Password strength meter bar changes color from red (low) -> yellow (medium) -> green (high)
- "Password (confirm)" field borders change to red when password doesn't match
- Add "password match/doesn't match" message below the confirmation password field

Success example: the user types a strong password (password strength bar changes color from red -> yellow -> green). Next, the user types a password confirmation that matches. No error is displayed, everything is green, and "Next" button is enabled.
![success](https://user-images.githubusercontent.com/45968869/131997046-58f7f7b7-86fa-42a9-8af5-b40e48180226.gif)

Failed example: the user types a partial password confirmation that doesn't match. When the user types a wrong character, "Password (confirm)" field shows red borders and a red error message is displayed. "Next" button stays disabled. The backspace key can be used to correct the typing error.
![fail](https://user-images.githubusercontent.com/45968869/131997053-fa345380-7bcc-4ed7-886c-1ab34a5f9012.gif)
