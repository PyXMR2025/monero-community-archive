# Original Description
This pull request provides support for mining [Salvium](https://salvium.io/) using algorithm `rx/0`. Included are

1. the details of the coin (algorithm, atomic unit size, block difficulty target, etc)
2. the details of the wallet adress prefixes and ports for each network type
3. the BlockTemplate parsing code required to make sense of the `protocol_tx` functionality of Salvium
