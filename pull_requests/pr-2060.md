# Original Description
![](https://i.imgur.com/v5YniBC.png)

https://imgur.com/a/EBhSh9j

- Theme color transition animations
- Touched almost every component, made sure they get their colors from `MoneroComponents.Style.*`
- Replaced background .jpg gradients with a `LinearGradient` for 'real' gradients + fallback when using the software renderer (low graphics mode) for the following components:
  - middlePanel
  - leftPanel
  - StandardDialog
  - wizardController
  - titleBar
- Rewrote titleBar.qml
- Introduced separate titleBar for merchant page
- Introduced folder with graphical effects; see `components/effects/`
- Replaced & removed a lot of icons in favour of `.svg`
- Introduced `MoneroEffects.ImageMask` to allow manipulating a `.svg` by opacity and colour at the same time.
- Theme definitions are in `components/Style.qml`
- The following text elements will transition text by default upon changing theme:
  - MoneroComponents.TextPlain
  - MoneroComponents.TextPlainArea
- Replaced all `Text {}` in favor of `MonerComponents.TextPlain {}` because `Text {}` by default has it's `textFormat` property set to `Text.RichText` which is downright dangerous, because it will render HTML which can be problematic/exploitable when rendering text from external sources.
- Some elements received a Shadow effect during white theme.
- Upgraded QtQuick imports to 2.9 (Qt 5.9.7)
- Refactored `DatePicker.qml` (history view), UX is now better.

**Please note**:

- `scaleRatio` will be removed in its entirety in a later PR.
