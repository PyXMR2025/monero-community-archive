Criterion benchmarks are straightforward, they are just regular benchmark crates within `benches/criterion/` that use `criterion`.

To run benchmarks:
```bash
# all benchmarks
cargo bench

# specific benchmark
cargo bench -p cuprate-criterion-json-rpc
```
