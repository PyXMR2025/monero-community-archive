# Original Description
Haven offshore now requires a **pricing_record** struct (size 192 bytes) to be present in the block_header. Therefore the blob size for a block_header has increased significantly.

This pull request increases the max supported blob size to 384 bytes - the first multiple of 96 above the required size for Haven offshore.
