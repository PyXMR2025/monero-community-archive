## koitsu | 2023-06-29T12:08:31+00:00
Also used by LCC (Litecoin Cash), PLSR (PulsarCoin), KEY (Keymaker), and MAZA (MazaCoin).  Refer to https://miningpoolstats.stream/ section "CPU - Various Algo" for examples.

Additionally, Zergpool supports mining some of these but not all: https://zergpool.com/site/mining?algo=minotaurx

Don't mistake MinotaurX for Minotaur (not same algo).

Source code that could be imported (I haven't checked license compatibility though -- edit: cpu-miner-multi is GPLv2, litecoincash is MIT):

* https://github.com/litecoincash-project/cpuminer-multi
