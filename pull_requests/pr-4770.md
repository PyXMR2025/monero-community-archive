# Original Description
Bandwidth constrained monerod node owners may want to know the limit-rate-up and limit-rate-down default values so that they can be set if needed before running the node, but --help does not display those values (only the -1 default). The only way to find out those values is by looking at the source code (currently 2048 & 8192 kB/s), or by running the node and then getting the value. 

Made changes to expose those values when ./monerod --help is entered.
