# Version: v0.17.1.3

# Release Notes
# Overview

This is the v0.17.1.3 minor point release of the Monero GUI software.
This is a recommended release that improves overall network perfomance.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.17.1.3)

Some highlights of this minor release are:

- Android support (experimental)
- Linux binary is now reproducible (experimental)
- Simple mode: transaction reliability improvements
- New transaction confirmation dialog
- Wizard: minor design changes
- Linux: high DPI support
- Fix "can't connect to daemon" issue
- Minor bug fixes
