# Original Description
My scripts used for generating code coverage report in a production ready (I hope) state. This includes an extensive documentation in the example script `utils/health/coverage-example-tdd.sh`. This will allow individual devs to generate a line coverage report locally, that will be matching the ones that I publish [online here.](http://cryptog.hopto.org/monero/health)

The scripts are meant to be run from Monero's root directory, except for the `utils/health/coverage-common.sh` which is only meant to be sourced.
