# Original Description
When the user creates a wallet with no password, the intention is to create a wallet with password disabled, because its easier and faster than having to type a password every time you open the wallet. The intention is not to create a wallet with blank password.

Therefore, no password prompt should be displayed when opening a wallet with no password.
