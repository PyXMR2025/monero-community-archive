# Original Description
20-30% faster dataset initialization on some CPUs:
- Zen3 (all)
- Zen2 without SMT (some of mobile Ryzen 4000 CPUs)
- Intel CPUs without HT (some Core i3 and i5)

New config option `init-avx2` controls it:

-1 = Auto detect
0 = Disabled (use old unoptimized code)
1 = Always enabled (only on CPUs that support AVX2)
