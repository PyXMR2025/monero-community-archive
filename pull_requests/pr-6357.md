# Original Description
Improved public node selection algo used in `Simple mode`/`---bootstrap-daemon-address auto`.

1. In-memory public nodes cache storing up to N (N=1000 by default) entries
2. If the cache doesn't have at least one good public node (having fails count == 0):
    1. Get public nodes from P2P
    2. Add new white public nodes to cache with initial fails count = 0
    3. Add new gray public nodes to cache with initial fails count = 1
    4. Truncate the cache to N public nodes, keep the nodes having lower fails count
3. Select most stable public node (sorted by fails count), if there are multiple - choose a random one
4. Send an RPC request:
    1. On fail - increase the node's fails count by 2 and go to step 2
    2. On success - decrease the node's fails count by 2 and go to step 4
