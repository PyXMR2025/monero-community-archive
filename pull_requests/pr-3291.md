# Original Description
Currently you have to click on a different tab to hide all the advanced subtabs and this feels clunky, especially if you're on a smaller screen where you then need to scroll in order to see all the tabs when they are uncollapsed. This allows the user to simply reclick on the advanced tab to collapse the subtabs.
