Changes:
- Hide wallet/node progress bar when sync is completed (this is the expected behavior for download/installation progress bars)
- Hide wallet progress bar when node is still syncing
- Add tooltips with instructions about wallet restore height and block height. Since setting a wrong restore height is a common problem, I suggest keeping these tooltips until mnemonic seed with embedded restore height is supported.
- Display wallet restore height. This will facilitate troubleshooting/user support, since all relevant data (wallet restore height, current block height, and connection type) will be displayed on the bottom left side.
- Display "Node" instead of "Daemon"
- When clicking, it opens Settings > Info page
- Display "Syncing..." on balance card when wallet progress bar is being displayed (sometimes the "Syncing..." message was not being displayed when wallet was still syncing)
