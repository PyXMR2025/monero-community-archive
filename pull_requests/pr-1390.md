## apertamono | 2018-05-11T11:42:57+00:00
Posting this here because I can read French, but it's probably a confusing issue for all languages: the %1 placeholder in `%1 blocks remaining: ` and `Synchronizing %1` does not represent a number of blocks, but rather the string `syncType`, which is either Wallet or Daemon. See the GUI code in components/ProgressBar.qml. The number of blocks remaining will be added after `%1 blocks remaining: `. Relevant code fragments:

``` 
    property string syncType // Wallet or Daemon
    property string syncText: qsTr("%1 blocks remaining: ").arg(syncType)
```

```
                progressText.text = syncText + remaining.toFixed(0);
```

```
            text: qsTr("Synchronizing %1").arg(syncType)
```
