# Original Description
Right now the description on the Mining page reads very haltingly. It has five short sentences that are broken up with line breaks because in compact view the words would run off the side.

This page needs dynamic resizing like the Check Payment page, which has a long sentence that automatically breaks down to the next line when it hits the edge of the page.

![mining](https://cloud.githubusercontent.com/assets/21302237/23571572/0ceae3a4-0038-11e7-9852-9ecfb04d8695.png)

The description on Mining could then be two paragraphs instead of five (this is new copy):

**Mining with your computer helps strengthen the Monero network. The more mining is done, the harder it is for the network to be attacked, and every little bit helps.**

**Mining also gives you a small chance to earn some Monero. Your computer will create hashes looking for block solutions. If you find a block, you will get the associated reward. Good luck!**
