# Original Description
Currently the restore height setting is done in a single text field that accepts either the blockheight or the date in format YYYY-MM-DD. This date format is not usual in many countries, and some countries use DD-MM instead of MM-DD.

Suggestion 1: use a slider
- Automatically updates the blockheight and date when user moves cursor
- Blockheight and Date fields could also be editable (user typing the value)

![image](https://user-images.githubusercontent.com/45968869/69710264-6414a180-10ff-11ea-9586-4ed6c09518c3.png)


Suggestion 2: use a text field + calendar
- Use a radio button to select whether use blockheight or date.
- If Blockheight is selected, use a text field.
- If date is selected, display a calendar (like the one used in Transactions page).
