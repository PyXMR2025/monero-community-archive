> EDIT: Only one NUMA node can access RAM directly. All others have to go through that one.

Just to better explain this. There is only one stick of RAM connected to each CPU. Epyc 7551 have 4 NUMA nodes and each NUMA node have 8 cores. Since only one RAM module is installed, only one NUMA node can access RAM directly. Other 3 NUMA nodes have to access RAM through first NUMA node which creates too much overhead and memory latency.
