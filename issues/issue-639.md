## ghost | 2017-04-01T23:40:38+00:00
Hi @x775 please feel free to help out! 

I'd recommend cloning the repo locally, making a new branch, copying the english (or other language if you prefer) translation from `src/monero-core/translations/monero-core_en.ts` to `monero-core_dk.ts` then opening it up and translating as necessary.

Then submit a pull request with a national flag in the appropriate size inside `src/monero-core/lang/flags`, amend the `src/monero-core/lang/languages.xml` file as necessary and voila!

It's actually easier than it sounds, and the existing code helps you a lot. Please try it out and if you get stuck just report back here.
