## hinto-janai | 2024-06-07T17:17:56+00:00
It's less about optimization and more about identifier safety, e.g. the following requires 1 ident modification when refactoring:
```rust
let x = &y[0];
let _ = x.a;
let _ = x.b;
let _ = x.c;
let _ = x.d;
```
but this requires 5 line changes:
```rust
let _ = &y[0];
let _ = &y[0].a;
let _ = &y[0].b;
let _ = &y[0].c;
let _ = &y[0].d;
```
It's harder to mess up when making changes in the 1st example.
