because the argument to the distribution sampler needs to satisfy the [UniformRandomBitGenerator](https://en.cppreference.com/w/cpp/named_req/UniformRandomBitGenerator) requirements. The following eliminates the error:
```c++
    struct gamma_engine
    {
      typedef uint64_t result_type;
      static constexpr uint64_t min() { return 0; }
      static constexpr uint64_t max() { return std::numeric_limits<uint64_t>::max(); }
      uint64_t operator()() { return crypto::rand<uint64_t>(); }
    } engine;
```
