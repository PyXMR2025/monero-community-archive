# Original Description
It's tax time and I need to report on my mining reward income sent to my wallet. For that, I need to know all the XMR I've received and on what dates. Currently, I can visually see that information by going to the "History" section in Monero GUI, but once in this view, I cannot copy the data out and there's no feature to export it either. It would be fantastic if this view (the "History") had an "Export" button to provide a CSV of the history data.
