# Original Description
It would be really amazing to have this project available to deploy on Single board computers like Raspberry Pi or Nvidia Jetson nano using [balena](https://balena.io). Computers like Intel NUC are also supported.

This will help users to directly use [deploy-with-balena](https://www.balena.io/docs/learn/deploy/deploy-with-balena-button/) button from README or from [balenaHub](https://hub.balena.io) and have it running with ease. They can even use balenaCloud dashboard to manage the fleet of miners using this project ex: change mining pool, enable, disable CUDA.

It requires having a Dockerfile, and I've tested this [here](http://github.com/iayanpahwa/monero-miner)

I am happy to help with the contributions required to balenafy xmrig 🙂
