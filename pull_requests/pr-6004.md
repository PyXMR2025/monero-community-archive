# Original Description
It got switched to hexadecimal when we went to 128 bit values

# Discussion History
## stoffu | 2019-10-24T01:43:11+00:00
Boost's cpp_int supports construction from a hex string (https://www.boost.org/doc/libs/1_56_0/libs/multiprecision/doc/html/boost_multiprecision/tut/conversions.html), so we can just do
```c++
       << "difficulty: " << cryptonote::difficulty_type(header.wide_difficulty) << std::endl
```
instead of introducing the new function `diff_to_dec`.
