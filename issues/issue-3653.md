# Original Description
To reproduce, make sure the GUI is in the foreground and that custom decorations are used.

Next click the Monero GUI icon in the taskbar.
Expected result is for the GUI to minimize to taskbar.
Actual result is no result (as if the program is trying to become the foreground application it already is).

I also noticed that when the minimize icon is clicked, the window just disappears instead of the minimizing to taskbar animation one would expect.
