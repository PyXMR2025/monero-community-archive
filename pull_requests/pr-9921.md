# Original Description
The intent is that you add "-D EXPECT_FUNCTIONAL_TESTS=1" to the targets that are run from CI, or for release preparation, so those tests aren't suddenly silently skipped if a dep is not found (which happened to me in TF because someone added a new python dep to monero).

By default, tests will still be silently skipped.
