# Original Description
Data formats aren't uniform across the globe. Therefore, it would make sense to add an option that would allow the user to choose his preferred date format. 

Initially suggested by c-789 [on reddit](https://www.reddit.com/r/Monero/comments/6wlzm1/help_out_with_testing_the_upcoming_gui_release/dm9p0me/):

>On the History tab, the dd.mm.yyyy date format may confuse those who are used to mm.dd.yyyy. An option to pick the date format would be helpful.
