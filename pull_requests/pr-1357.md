# Original Description
- refactor some platform dependant code. At least for me is easier to read this way.

- add isDesktop, to differenciate between desktops OSes and mobile OSes. It is useful mainly because android is recognized by qt as linux too.

- add a initial cmdline parser. Move some qDebug  and force style fusion for desktop platforms so don't pollute the output of the parser. It will be useful in the future :smiley: 

- move monero stuff after app constructor + cmdline parser. (We don't want to initialize anything if we're running, for example monero-wallet-gui --version)

- force platform xcb on desktop linux, because on newer systems qt will try to use platform wayland and this isn't supported by our building system yet.

- fixed reported height and width, because it was right only when height > width.
