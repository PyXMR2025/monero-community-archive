# Original Description
This PR has turned into a pretty big clean up of the consensus code, specifically `cuprate-consensus` the crate which holds the services that handle verification. `cuprate-consensus-rules` should be mostly unchanged.

This PR also fixes some issues with our consensus code, regarding cross-block batch verification, which is batch verifying txs together from multiple blocks. 

I have decided to remove cross-block batch verification for now, in the future we can look for better solutions like staged syncing.

Sadly by not cross-block batch verifying we will lose out some performance when syncing, I think this is ok for now as we will have a fast sync mode where verification will be skipped for known block hashes which would a lot quicker anyway.

I have also decided to remove our RPC scanner, as although it was extremely quick for what it was doing, syncing a Cuprate node should be even faster so I can't see a reason to keep it around. Without cross-block batch verifying the RPC scanner would be extremely slow as well.

If you are looking to use Monero consensus rules it's recommended you try to integrate `cuprate-consensus` and fall back
to `cuprate-consensus-rules` if you need more flexibility.

Was thinking it would be slower since `.copied()` copies the full `Option<u128>` (size_of = 32 instead of 16) although it seems like the extra ref indirection may be worse:
```rust
use std::hint::black_box as bb;
use std::{collections::VecDeque, time::Instant};

fn main() {
    loop {
        let mut cumulative_difficulties = VecDeque::from([0]);
        let difficulties = vec![0];

        let now = Instant::now();
        for _ in 0..1_000_000_000 {
            // let _ = bb(old(&mut cumulative_difficulties, &difficulties));
            let _ = bb(new(&mut cumulative_difficulties, &difficulties));
        }
        println!("{}", now.elapsed().as_secs_f32());
    }
}

fn old(cumulative_difficulties: &mut VecDeque<u128>, difficulties: &Vec<u128>) {
    let last_cum_diff = bb(cumulative_difficulties.back().copied().unwrap_or(1));
    cumulative_difficulties.push_back(last_cum_diff + *difficulties.last().unwrap());
}

fn new(cumulative_difficulties: &mut VecDeque<u128>, difficulties: &Vec<u128>) {
    let last_cum_diff = bb(cumulative_difficulties.back().unwrap_or(&1));
    cumulative_difficulties.push_back(last_cum_diff + difficulties.last().unwrap());
}
```
