# Original Description
Remove `-X POST` from calls to curl. It doesn't do anything good and now the commands are shorter.

[Article by curl author warning against the unnecessary use of -X](https://daniel.haxx.se/blog/2015/09/11/unnecessary-use-of-curl-x/)

# Discussion History
## 00-matt | 2020-05-24T21:20:52+00:00
The `-X POST` was not unnecessary here as there is no `-d` flag, however I don't think this should be a POST anyway since we just get data. (I tested with GET and POST, you get the same response)
