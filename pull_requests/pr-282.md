# Original Description
Could be useful for old non-deterministic wallets to be able to dump the account to import into MyMonero. For deterministic wallets, this will just be a hex representation of the seed, so it's bit redundant, but OTOH simplewallet isn't exactly overflowing with commands, so I don't see it as an issue.
Tested on W7-64.
