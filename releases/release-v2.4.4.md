# Version: v2.4.4

# Release Notes
 - Added libmicrohttpd version to --version output.
 - Fixed bug in singal handler, in some cases miner wasn't shutdown properly.
 - Fixed recent MSVC 2017 version detection.
 - [#279](https://github.com/xmrig/xmrig/pull/279) Fixed build on some macOS versions.
 - For Windows builds, libuv updated to 1.18.0 and libmicrohttpd to 0.9.58.

For macOS please use [Homebrew](http://www.brewformulas.org/Xmrig) to install.
