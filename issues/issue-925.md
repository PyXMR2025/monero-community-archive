# Original Description
It is suboptimal to show a QR code which is supposed to be scanned by customers while displaying the wallet balance in the top of the left sidebar. Even if there is a button to hide the sidebar, it should be hidden by default in the Receive screen. Alternatively make it configurable whether to show the Balance at all or only in a dedicated tab (eg., My Balance tab). Maybe the History tab is already suffient for some people.
