# Original Description
With the completion of Zero to Monero (see [here](https://getmonero.org/library/Zero-to-Monero-1-0-0.pdf)), the urgency of this has decreased. Having said that, we should like to provide a series of short standards (see the [CryptoNote standards](https://cryptonote.org/standards/) as an example) explaining the usage of each individual element in Monero and how it all comes together.

This is a very descriptive project, all about documentation, rather than a research project.
