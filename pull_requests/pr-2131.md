# Original Description
This is @moneromooo-monero's wallet generator (https://github.com/moneromooo-monero/monero-wallet-generator) adapted for getmonero. It will be available at `getmonero.org/generator`.

Changes:

- changed structure to make it more mobile friendly
- removed some unused css 
- added getmonero's headers for SEO  and opengraph views

I touched the file as less possible to make reviewing easier. It could be cleaned up further. I stopped when realized there is too much stuff entangled with each other (printable page, mobile, etc) and would take too long. Probably not worth it.

The file assumes there is a signed version of the page hosted in the same folder and signed with @binaryFate's key.
