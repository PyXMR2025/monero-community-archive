# Original Description
This adds the block downloader, which is way more complicated than its name suggests.

The block downloader's job is to find a higher chain to follow and follow it, requesting blocks from peers and guaranteeing the blocks are yielded in order.

More details are on the docs for the `BlockDownloader`
