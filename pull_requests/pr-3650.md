# Original Description
This can be used by merchants to create payment requests for products or services, or to create a QR code for asking donations in a website/donation campaign. A QR code and a payment request text (URI) are created containing the data inserted in the text fields (both can be copied to the clipboard and easily sent through email, instant messenger, chat, etc.). 

"Description" field (`tx_description` in URI) is used by Monero GUI as transaction description when the QR code is scanned in Send page. 

"Your name" field (`recipient_name` in URI) is used by Monero GUI when the QR code is scanned in Address book page (users can create a new contact without having to type address/name).
