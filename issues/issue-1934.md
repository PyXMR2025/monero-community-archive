# Original Description
Hello,
On the website https://www.getmonero.org/get-started/what-is-monero/ there is an outdated statement:

"Unlike selectively transparent alternatives (e.g. Zcash), Monero is the only cryptocurrency where every user is anonymous by default."

Although it may have been true when written, Monero can no longer claim to be the "only" cryptocurrency that is anonymous by default. It may be the most trusted, time-tested, and peer-reviewed privacy crypto but it is no longer the "only" one. Could this be updated or changed by someone to reflect reality? Probably the simplest thing to do would be to change it to

"Unlike selectively transparent alternatives (e.g. Zcash), Monero is anonymous by default."

But it can be changed to anything as long as it is not a falsehood.

Thank you!
