# Original Description
Reported by scoobybejesus on reddit:

-If there's room down in the lower left, which it appears there usually is (see last item below), it seems wise to have a "Network Status" and a "Wallet Status," since they are two different things that both affect the extent to which the wallet will reflect an accurate balance.

----------------

To add, it would probably be best if we have two individual status bars, i.e., one that would display the wallet (sync) status and one that would display the blockchain (sync) status). This was also suggested by other people (e.g. hyc and gingeropolous).
