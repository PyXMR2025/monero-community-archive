# Original Description
For example, in the following usage scenario:

```yaml
# docker-compose.yml
services:
  tor:
    hostname: "tor.docker"
    image: any/tor/image

  monerod:
    image: any/monerod/image
    depends_on:
      tor:
        condition: service_healthy
    entrypoint: monerod
    command:
      - "--proxy=tor.docker:9050"
```

Error log:

```
ERROR	net.p2p	src/p2p/net_node.inl:936	Failed to parse proxy: tor.lan:9050 - net::error_category:9
```
