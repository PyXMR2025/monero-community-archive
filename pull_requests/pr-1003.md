# Original Description
After @radfish's changes, I've rewritten the CMakeLists.txt, Makefile and README.md to add support for ARMv8 compilation.

Also added flags for workarounds of issues on the Cortex-A53 that affect 64-bit MAC operations and internal paging 

CMakeLists should also try to select the most appropriate flags for the FPU and floating-point ABI for ARMv7 devices

NB ARMv8 has the FPU and NEON SIMD 'always on' which is why -mfpu and -mfloat options cease compilation
