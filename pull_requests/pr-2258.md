# Original Description
Requires #2257 

This PR adds support for Tails and [its persistent volume feature](https://tails.boum.org/doc/first_steps/persistence/index.en.html).

At launch it will:
1. Detect Tails OS
    - expose bool `isTails` to QML
2. Detect if a Tails encrypted and persistent volume is mounted
    - If not, warn the user about settings that will not be saved
3. Ask if the user wants to point the GUI to this persistent volume:
    - expose bool `tailsUsePersistence` to QML
    - Write log files to `~/Persistent/Monero/monero-core.conf`
    - Create and use wallet directory: `~/Persistent/Monero/wallets`
    - Write the `monero-core.conf` configuration file to `~/Persistent/Monero/`
    - Register a persistent desktop startup entry

To sum up, this PR detects Tails and sets the correct paths.

When Tails is detected, but persistence is disabled, show a warning:

![](https://i.imgur.com/sXZujXI.png)

When Tails is detected with persistence enabled, ask the user to write to it:
