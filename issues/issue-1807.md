# Original Description
https://www.getmonero.org/resources/moneropedia/stealthaddress.html

"They allow and require the sender to create random one-time addresses for every transaction on behalf of the recipient."

"allow" is redundant when "require" is in the same sentence.

Succinct: "The sender is required to to create random one-time addresses for every transaction on behalf of the recipient."

I also recommend adding more technical information on how an address is randomly created in a way that still links to the primary address. I know it has something to do with mod.
