# Original Description
This PR scans for `.keys` files in the Monero accounts directory, displays them in a Listview. 

Old situation was [naively assuming directories](https://github.com/monero-project/monero-gui/blob/d2c47606ca8446e58ef122117f8f84b578691bab/wizard/WizardOpenWallet1.qml#L45-L55) in `~/Monero/wallets/*` were valid wallet paths, this actually checks for the presence of `.keys`. In addition, it can detect what network type a wallet is and the GUI switches to that upon opening the wallet.
