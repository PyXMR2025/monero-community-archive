# Original Description
Experimental mobile layout + ios settings
Requires https://github.com/monero-project/monero/pull/1950

# Instructions

First you need to build IOS version of boost and crypto libs:
https://github.com/x2on/OpenSSL-for-iPhone
https://github.com/danoli3/ofxiOSBoost

### Build libwallet
`./ios_get_libwallet.api.sh`

### Build GUI
When iOS version of libwallet is sucessfully built you can start building the GUI. 

First you need to make sure that you've installed the QT iOS dependencies (~/Qt/5.7/ios). You can install it from MaintenanceTool.app in your Qt folder if you haven't installed it from start. 

### Create a XCode project
Fire up a terminal 
Add ~/Qt/5.7/ios/bin/ to your $PATH
in monero-core folder:
`qmake -spec macx-xcode monero-wallet-gui.pro`

More info about Qt and iOS. 
http://doc.qt.io/qt-5/ios-support.html
