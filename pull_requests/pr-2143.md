# Original Description
This PR adds [Coin Wallet](https://coin.space/monero-wallet/) a non-custodial multicurrency crypto wallet to the download page.

* Coin Wallet was launched in 2014
* Monero support was added [July 2021](https://github.com/CoinSpace/CoinSpace/releases/tag/v4.0.0)

Main repo is here: https://github.com/CoinSpace/CoinSpace

Client code is completely open source. Links to monero part:
* https://github.com/CoinSpace/cs-monero-wallet
* https://github.com/CoinSpace/monero-core-js
* https://github.com/CoinSpace/monerolib

Almost two year have passed since the last request to add Coin Wallet to download page: https://github.com/monero-project/monero-site/pull/1725
