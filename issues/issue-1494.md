# Original Description
AV software typically flags Monero as 'unintended' software (predominantly due to a miner being included). New users are typically affected by this and it may scare them. Whilst we have a proper FAQ on the matter, it would probably be worthwhile to add a small pop-up to inform the user before they download the software. 

# Discussion History
## erciccione | 2021-03-01T08:12:54+00:00
We already have a warning in the Downloads page. Actually, two, one for the CLI and one for the GUI: `Be aware that some antiviruses and firewalls may flag the Monero executables and archives as malware. More info in the FAQ.(link)`.

Maybe we could add a header to make it more visible?
