## scottAnselmo | 2019-11-09T16:42:21+00:00
I'd like to add an item discussing using [OpenCollective](https://opencollective.com/) to make donating to Monero Project easier (this would not be a replacement to anyone straight up donating XMR to the general fund or CCS). Donations would be in fiat which wouldn't prevent things like [what happened with Tor fiat donators](https://portswigger.net/daily-swig/torservers-tails-and-riseup-european-donors-details-now-in-the-hands-of-the-german-police), but it would potentially help wean the projects dependencies on whales because people could donate $5/month, etc w/o thinking to the project. The goal would be the OpenCollective owners would convert it into Monero and deposit into whatever specified by the designated Monero folk. Rehrar mentioned at the last Coffee Chat something to the effect of he would potentially be interested in exploring doing this for MRL. I think we should explore it, but would like thoughts from others
