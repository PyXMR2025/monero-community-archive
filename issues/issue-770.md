# Original Description
The download page can be quite ambiguous for new users as it does not make reference to a wallet anywhere on the page in relation to the GUI and CLI applications. 

I know the GUI and CLI are much more than wallets but majortiy of users who visit this site, well I would presume, are there to download a monero wallet. 

An example of a minor improvement:
The links to download such as 'Windows, 64-bit' should say 'Windows, 64-bit Graphic User Interface.' As well as specify that this application can be used as a wallet somewhere.
