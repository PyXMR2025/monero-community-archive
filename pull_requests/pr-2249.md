# Original Description
Add ELPP_OS_OPENBSD macros to easylogging++.h so that it will build on
OpenBSD.

I made this change to get monero to build on OpenBSD 6.1 on x86_64 tonight. OpenBSD lacks the execinfo.h header file, so these changes will bypass including it on OpenBSD.

I'm not 100% sure that more changes are not required because my machine runs out of memory on core_rpc_server.cpp. I'm going to try again with more RAM, possibly on a virtual machine.
