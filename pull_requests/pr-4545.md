# Original Description
This PR contains 2 patches for the Windows daemon:

- Add detection of OS versions newer than Windows 7 when calling **--os-version** argument
- Fix terminal colors on newer versions of Windows 10 - see issue https://github.com/monero-project/monero/issues/1934

This was tested working on Windows 10 build 14393.
