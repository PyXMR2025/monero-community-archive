This CI builds release binaries. Usage is:
1. Actions
2. Release Builds
3. Run workflow
4. Enter commit to use
5. Wait for builds, download uploaded `zip` files
6. Extract for `cuprated-$VERSION-$OS-$ARCH.{tar.gz,zip}`

Tested here: https://github.com/hinto-janai/labeler-test/actions/runs/13778547686.

Contents for all releases are:
```
cuprated-0.0.1-$OS-$ARCH/
├── cuprated
├── Cuprated.toml
└── LICENSE
```
`LICENSE` is AGPL-3.0. Linux will include `cuprated.service` after https://github.com/Cuprate/cuprate/issues/396 is fixed.
