# Original Description
This crate/lib will implement D++ using the `Service` trait to be generic over clients.

To fully implement D++ you need knowledge of and control over the tx pool which would be pretty annoying to abstract over so this crate will contain a minimal D++ router that will just handle tx routing and nothing to do with keeping track of tx states etc.

This will still have a D++ txpool abstraction for completeness which Cuprate will use.
