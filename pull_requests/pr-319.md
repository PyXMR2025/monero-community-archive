# Original Description
Multiple edits for copy on Create/Restore wallet wizard. Most simplified and shortened the existing text, others clarified what was a bit confusing ("enter your 25 word private key"), while others normalized more terminology (no longer calling wallets "accounts"). 

Probably the biggest change was to the titles of the pages. The previous titles tried to lead the user through the process ("A new wallet has been created for you"), but this often served to detract from the purpose of the page, for example the page "A new wallet has been created for you" was the page where the user was supposed to give their wallet a name. Now all the titles are simple and uniform ("Give your wallet a name" - "Give your wallet a password") and any necessary details were added below the titles.
