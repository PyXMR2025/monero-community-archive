# Original Description
### What
Adds a `constants` and `statics` module to `cuprated`.

- `constants` contains general constants used by  `cuprated`, currently it contains version strings roughly equivalent to [`version.h`](https://github.com/monero-project/monero/blob/a1dc85c5373a30f14aaf7dcfdd95f5a7375d3623/src/version.h)
- `statics` contains global `static` data used throughout `cuprated`

Some data (e.g. `const DEBUG: bool`) will be available in [`cuprate_constants`](https://github.com/Cuprate/cuprate/pull/280) so it isn't defined in `constants`.

### Why
- `statics::START_INSTANT_UNIX` is needed by RPC for [`get_block` -> start_time](https://www.getmonero.org/resources/developer-guides/daemon-rpc.html#get_block)
- `constants::VERSION_BUILD` is needed by RPC for [`get_block` -> version](https://www.getmonero.org/resources/developer-guides/daemon-rpc.html#get_block)
