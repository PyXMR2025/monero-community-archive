## hyc | 2019-11-19T08:46:25+00:00
This is only constant time if a particular CPU has a barrel shifter. Otherwise it can be linear in `i`. For constant time it would be better to do 1-bit shifts every time, like so:
````
while (i < 64) {
  amountb[i++] = val & 1;
  val >>=1;
}
````
