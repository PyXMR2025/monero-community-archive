# Original Description
This PR adds the possibility to translate into some new languages:

 - Urdu
 - Persian (Farsi)
 - Zulu
 - Kurmanji (Kurdish)
 - Bengali (Bangladesh)
 - Gaelic (Irish)

The flags of Bangladesh and South Africa were already present, i added the other ones.

After this PR will be merged, i will update [Pootle](https://translate.getmonero.org), so that translators can start work on these languages. For now all the new .ts files are completely untranslated

All new languages are commented out from languages.xml and they will stay not selectable until they are translated for at least the 80%. Then i will activate them (from this account or @Monero-Pootle )
