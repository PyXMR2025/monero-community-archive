# Original Description
Currently we don't ask users to write down the restore height when creating a new wallet from hardware device. Restore height is displayed on the last page (summary page), but the user is not asked to write it down. Probably this is the main reason why users usually report that they don't know the restore height of their hardware wallets when they are restoring.

Changes:
- Add "Write down your restore height" page, which is displayed when creating a new wallet from hardware
