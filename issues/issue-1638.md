# Original Description
The Merchants page lists businesses and services that accept Monero in exchange for said services/products. The 'Tools' section is an exception, as it lists tools like block explorers, which accept Monero donations.

I think the merchants page is not the best place for these services. I suggest to remove the 'Tools' section from the merchants page and instead create a dedicated section somewhere else.
