# Original Description
Since #1047 was merged, my Ubuntu 14.04 build environment consisting of four GCC compiler versions (4.8.5, 4.9.4, 5.4.1 and 6.2.0), all managed by update-alternatives, is able to compile monero only with GCC 4.8.5. After switching to any other compiler, there is a massive amount of unresolved dependencies when linking.

This is an attempt to preserve the fix to #1026 while not breaking build environments not affected by that issue. This may also fix #1064.
