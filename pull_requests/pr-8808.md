It looks like a node would relay a tx that has `m_no_drop_offense` set to true because it continues executing past the if statement.

Looks like you'd want something more like:

```cpp
if (!m_core.handle_incoming_tx({tx, crypto::null_hash}, tvc, tx_relay, true))
{
        if (tvc.m_no_drop_offense)
        {          
            LOG_PRINT_CCONTEXT_L1("Tx verification failed with a no drop offense");
            continue;
        }
        LOG_PRINT_CCONTEXT_L1("Tx verification failed, dropping connection");
        drop_connection(context, false, false);
        return 1;
}
```
