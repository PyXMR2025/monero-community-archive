# Original Description
Required for #2637

Currently LineEdit uses an Item{} component while LineEditMulti uses a ColumnLayout{}. 

This PR modifies LineEdit component to use ColumnLayout{}. This is useful because we can set its properties (like margins) with the same values of a neighbor LineEditMulti component.

This modification requires some changes to LineEdit components in Transfer and WizardWalletInput.

Other changes:
- Removing unused inlineIcon
- inputHeight changed to 39 (the same as LineEditMulti)
- Add paste button
