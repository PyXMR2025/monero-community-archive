# Original Description
The CLI wallet has a `--restore-from-spend-key` option which will restore a deterministic wallet from only the private spend key. This functionality would be nice to have in the GUI wallet too. This PR will generate and use a deterministic viewkey if a private spend key is supplied but a private view key is not when restoring a wallet
