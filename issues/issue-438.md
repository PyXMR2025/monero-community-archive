Monero's 6th "Moneroversary" (birthday/anniversary) is on April 18. It falls on a Saturday this year, so we have a ton of flexibility to host events all day. This issue shall serve as a discussion to brainstorm ideas for the event.

The 5th Moneroversary included the following events (#324):

![image](https://user-images.githubusercontent.com/12520755/74106758-d60d5c00-4b2e-11ea-964d-e7153ee2bf64.png)

Let's make the 6th Moneroversary even better! We should consider the following:

1. ~~Physical meetups and check-ins~~
2. Games
  a. Kahoot Trivia (I have a license)
  b. Secret Hitler (open-source software available)
  c. Jackbox.TV (@rehrar has a license)
  d. Meme competition
  e. Something with EtherDraw
  f. Drawful
3. Coffee Chat discussion
4. Monero spend activities (#435; @xenugrande)
5. Puzzles
6. Barolo toast

Please comment below with ideas. We can put a schedule together as the ideas start to take on some traction.
