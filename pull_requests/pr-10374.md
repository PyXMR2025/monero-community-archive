# Original Description
According to the C standard, right-shift for negative left-hand-side operands is implementation-defined, and left-shift for negative left-hand-side operands is undefined. This makes it so that `sc_check()` is standards-compliant, or relies on GCC implementation-defined behavior only when GCC is present. This change is a bit pedantic since all supported platforms use this behavior AFAICT, so this doesn't really change anything.
