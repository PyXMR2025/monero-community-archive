# Original Description
Was requested on Reddit.  I thought it would be ridiculous and too complex but it turned out to be pretty easy.

Tested on Win7 and Linux (I have no Win10 anywhere to test).  No MacOS support yet, but stubbed out so it shouldn't crash.

May be able to roll back some sections to c++11 but everything works apparently the same with c++17 anyway.

If `WITH_PAUSE_PROCESS` is off the resulting code is identical to without this patch (everything in ifdef wrapping).  Including no added libs and c++11 compatible.

Configuration is such as `"pause-on-process": ["/usr/bin/vlc"],` you can list many and it's substring case insensitive of the entire commandline on Linux including arguments, or the fully qualified EXE path on Windows (no arguments).
