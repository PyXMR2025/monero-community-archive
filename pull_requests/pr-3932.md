## stoffu | 2018-06-05T15:14:11+00:00
Aside from the default-to-trusted-or-untrusted question, I'm a bit concerned with this idiom of `boost::optional<uint32_t>` which seems to make the code look more complicated than necessary. Can't it be simply like this?

```c++
  if (args.size() == 2)
  {
    if (args[1] == "trusted")
      m_trusted_daemon = true;
    else if (args[1] == "untrusted")
      m_trusted_daemon = false;
    else
    {
      fail_msg_writer() << tr("Expected trusted or untrusted, got ") << args[1];
      return true;
    }
  }
  else
  {
    m_trusted_daemon = false;
    try
    {
      if (tools::is_local_address(m_wallet->get_daemon_address()))
      {
        MINFO(tr("Daemon is local, assuming trusted"));
        m_trusted_daemon = true;
      }
    }
    catch (const std::exception &e) { }
    success_msg_writer() << boost::format("Daemon set to %s, %s") % daemon_url % (*m_trusted_daemon ? tr("trusted") : r("untrusted"));
  }
```

If the current idiom is to be used, I think this line should be `m_trusted_daemon = *trustedness`.
