# Original Description
These changes aim to make the homepage a better source of info for the visitor. Right now the main section is "Why Monero is different", which is not something we should explain to somebody visiting the homepage (especially for the first time, different from what?). The point of these changes is to give immediately all the resources necessary to get started with Monero and, equally important, to easily find the info they might be looking for on getmonero.

I feel that the website is not seen as a point of reference and i think part of the problem is that it's not easy to find the info people need. This PR is part of the efforts of making the website easier to consult and the pages more interconnected.

The experience i aim to give is:

People open the website and they immediately see the promo video, which is a great first impact resource. If they know nothing about Monero, they will probaly click the button "What is Monero?" and they will be taken to the page where they can find more info.

If they are already somehow familiar with Monero, they will probably scroll down, where they will be able to directly find the info they are likely looking for: Download a wallet, get some coins, places where to spend their coins and a link to the FAQ, so if they have doubts, they can resolve them right away.

If they want more detailed info or are developers looking for instructions, we link to the guides and the library. If they are interested in contributing they can go to the "Contributing" page (which needs some big improvements).

In this PR i also removed the "Using" page, since those information are now in the homepage and i used the pictures in the new home. @binaryfate it will be probably necessary to create a redirect so that people visiting https://web.getmonero.org/get-started/using/ will be redirected to the homepage. I don't think many websites link to that page, but better not to risk to give a 404.
