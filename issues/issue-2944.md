# Discussion History
## selsta | 2020-06-11T17:19:37+00:00
Solo mining is more like a lottery. It will take you on average 550 days to find a block with your hashrate. If you are lucky, it will be sooner.

If you want more consistent payout I would recommend you a mining pool.

Also note that mining with 2kH/s is more for supporting the network, don’t expect it to be profitable.
