# Original Description
Many users are looking for a new wallet after their current wallet software, e.g. Exodus, did not update in time for the Fluorine Fermi hard fork. GetMonero.org should not point users to another wallet that is also currently not working.

Once MyMonero is confirmed to be working with the hard fork, it can be added to Downloads again.
