# Original Description
The [PoWER](github.com/monero-project/research-lab/issues/133) proposal may exclude smaller-input TXs from the PoW relay requirement. To allow wallets to skip implementing PoWER, the transaction builder code should enable an option to limit the number of inputs to select.
