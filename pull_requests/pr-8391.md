# Original Description
Print usage for these functions when given 0 arguments. All functions that expect arguments do this, except for these 4.

```
transfer
locked_transfer
locked_sweep_all
sweep_below
```

BEFORE:
```
[wallet 48hint]: transfer
Error: wrong number of arguments
```
AFTER:
```
[wallet 48hint]: transfer
Error: usage: transfer [index=<N1>[,<N2>,...]] [<priority>] [<ring_size>] (<URI> | <address> <amount>) [<payment_id>]
```
