# Original Description
This page is displayed in advanced mode when a wallet is being created or restored.

Changes:
- Changed title from Daemon settings to Node settings (to be consistent with Settings > Node page)
- Added FontAwesome icons for local node (home icon) and remote node (cloud icon), the same used in Settings > Node page. 
- Added FontAwesome cloud icon for the bootstrap node, to inform that it is a remote node.
- Moved settings to the bottom (consistent with Settings > Node page)
- Added subheaders "Local node settings" and "Remote node settings".  Previously it wasn't clear that the bootstrap node was an option of the local node settings.
- Added question mark icons with instructions and tips (blockchain download requires minimum of 90 GB, SSD faster than HDD to sync blockchain, what is a bootstrap node, how to find a remote node)
