# Original Description
I've repeatedly found myself with instances of questions/issues/proposals that don't fit into the categories of bitmonero, kovri, monero-site, and monero-forum. A few of these issues have been posted to both the forum and IRC but I've had little-to-no response and it is a bit discouraging - though I think the lack of response is understandable considering the nature of forum software and IRC in general (so, no hard feelings).

I propose a centralized repository that will cover all organizational issues unrelated to software development. Something that will give us a chance to easily review and contribute to project ideas in a memorable setting. The issues covered would also include all public DevOps-related concerns and _possibly_ all future [ideas](https://forum.getmonero.org/6/ideas) of which are currently only accessible through the forum.

Note: if anyone hasn't visited the [ideas](https://forum.getmonero.org/6/ideas) section in a while, I think that there are some good all-across-the-spectrum ideas that deserve some more attention.
