# Original Description
Achieved about 10% speed up for core_tests, by reusing the `crypto::key_derivation` variable, which was previously being needlessly regenerated for exactly the same parameters. In order to achieve this, I had to refactor `cryptonote_format_utils.cpp` a bit, so that the dependencies, namely `crypto::key_derivation` and `crypto::public_key`, may be generated independently. The refactoring also allowed to get rid of two duplications.
