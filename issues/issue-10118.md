# Original Description
https://github.com/monero-project/monero/blob/b591866fcfed400bc89631686655aa769ec5f2dd/contrib/epee/include/net/http_server_handlers_map2.h#L146-L157

The cited block of code attempts to load the request via `portable_storage`, whose JSON parser expects the first character to be an `{` (despite the possibility of it being `[` for a valid JSON encoding). It then proceeds to read fields from the object as if it was an object, despite the fact the [JSON-RPC 2.0 standard allows it to be an array of objects](https://www.jsonrpc.org/specification#batch) (each their own JSON requests).

This forces what could be one HTTP request to unnecessarily be several.
