# Original Description
During some builds I noticed that `sha256sum` was failing because it expects two spaces between the filename and hash, _and_ this was not causing `docker build` to fail.

This PR seeks to fix those `sha256sum` checks, as well as add `set -ex` to catch this, and other, errors in the future.
