# Original Description
All the essential `monerod` documentation, one quick place for docs without leaving your terminal:

- Includes `Options` and `daemon_commands`
- Comprehensive list of all `JSON RPC` methods
- Detailed `Example` section
- Code blocks included for clarity

This allows us to efficiently direct someone to a specific section of the man page, rather than vaguely referring them to "somewhere in the middle of the --help subcommand."

The man page will be bundled with the tarball releases, making it as simple as running man `./docs/man/monerod.1`.
