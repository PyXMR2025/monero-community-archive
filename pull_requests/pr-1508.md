# Original Description
Lazy loading makes web pages load images only when they are actually
needed (when they need to be displayed to the user), thus saving
resources for both host and users and improving performances for the
latter.

| Page | Page weight before (load time) |  Page weight after (load time) | Space savings (time savings)
|---|---|---|---|
|index.html | 474 kB (546 ms) | 214 kB (373 ms) | -54% (-31%)
| get-started/what-is-monero/index.md | 320 kB (548 ms) | 246 kB (525 ms) | -23% (-4%)
| get-started/accepting/index.md | 254 kB (747 ms) | 193 kB (536 ms) | -24% (-28%)
| downloads/index.md | 397 kB (595 ms) | 243 kB (383 ms) | -38% (-35%)

(tests ran locally)

This should work our of the box on Firefox and Chrome.

https://developer.mozilla.org/en-US/docs/Web/Performance/Lazy_loading
