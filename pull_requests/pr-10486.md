# Original Description
`get_tx_key`, `get_tx_proof`, `get_spend_proof`, and `get_reserve_proof` were available in `--restricted-rpc` mode.

These methods expose transaction secret keys or generate wallet ownership/spend proofs, so they should be restricted like other non-view-only wallet RPC commands.
