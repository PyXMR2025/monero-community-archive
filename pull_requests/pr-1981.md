# Original Description
Cannot be merged right now due to licensing restrictions - submitting this one for visibility & comments.

So far this PR gets rid of the old attempts at concurrency (just to give you idea about their quality - old sync time was 36 minutes at 1.8 - 2.0 load (out of 4.0), new sync time is 43 minutes at 1.0 load) and implements a proper data feed.

This PR demonstrates how I will do the rest of the code. Shared states will be exported to their own objects where possible, or synchronised after processing where it is not. Since the multithreaded code is much more complex now, I opted to catch the exceptions and rethrow them verbatim rather than just using an error flag.
