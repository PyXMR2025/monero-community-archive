# Original Description
Since Monero is private, users can reuse addresses without compromising privacy. Therefore, Monero GUI wallet should incentivize users to use the `Address book` feature.

After completing a transaction, the `Monero sent successfully` pop up should have two buttons:
- `Ok`
- `Add 8BgB Yrn5 ... to Address book`

When the second button is clicked:
- Either open `Add an address` page with the address prefilled and autofocus in the Description field,
- or open a pop up with the Description field (add a name) for the address book entry. After clicking Ok, the user should be redirected to the Address book to see the entry that has just been created.
