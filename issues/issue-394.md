# Original Description
Hi,
This is a feature request and I'm hoping that @Jaqueeee in particular will see this. I would like it if you could switch between multiple wallets using tabs.

I think that usage of multiple wallets will be typical with monero because one does not want to share a monero address with multiple third parties. Also if outfits like coinbase are demanding viewkeys one will ordinarily want to transfer one's money over to a more private wallet as soon as it's received.
