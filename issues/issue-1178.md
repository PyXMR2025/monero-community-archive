# Original Description
There is a lot of talks on i2p integration. If one day an i2p router is integrated into Monero, it should be possible to use an external i2p router.

I run i2p with an open port (for years because I need it). However my band width is limited and I prefer to avoid running two routers at the same time.

Monero should use its internal i2p router by default, but there should be a possibility to disable it and connect to an external one.
