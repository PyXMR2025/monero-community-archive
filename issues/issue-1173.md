## moneroguides | 2022-05-07T15:44:29+00:00
"Monero uses the unspent transaction output (UTXO) model for handling transaction verification.
 
Outputs, also known as 'owned outputs', represent some amount of Moneroj which has been authorised by one key pair to be spent by another.

Each transaction produces 3 outputs. The amount to be transferred, a transaction fee which is paid to miners for the verification process and change.

Even if the entire sum is consumed by the fee and intended output, there will still be an output of 0 for change, to make it indistinguishable from other transactions."
