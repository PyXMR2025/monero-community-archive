# Original Description
Since we're just counting txs, there's no reason to deserialize all the blobs.
Add a new get_transaction_pool_stats RPC that just returns the statistics.

Original code:
```
time ./monerod print_pool_stats
511 tx(es), 128702430 bytes total (min 13254, max 258312, avg 251863)
fees 148.600494060000 (avg 0.290803315185 per tx, 0.000001154605 per byte )
0 not relayed, 0 failing, 462 older than 10 minutes (oldest 3 hours ago)


real    0m12.368s
user    0m1.720s
sys     0m0.733s
```
New code:
```
time ./m2 print_pool_stats

real    0m0.006s
```
