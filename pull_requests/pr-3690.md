# Original Description
Requires #3687.

If the clipboard has a QR code image and the user paste (Ctrl+V) inside the text input, the QR code URI will be parsed as text. This feature is useful if someone sends you a payment request as a QR code image and you can't scan it (using mobile, for instance).
