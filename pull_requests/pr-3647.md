# Original Description
**Simple mode:**
This mode starts monerod daemon as a "remote node seeker" to search for remote nodes on Monero P2P network.
![image](https://user-images.githubusercontent.com/45968869/127620754-90599000-aed7-4762-9fe2-b8ef21251d02.png)
![image](https://user-images.githubusercontent.com/45968869/127359455-09aab126-5861-40aa-82ca-dd9766384b42.png)

**Simple mode (bootstrap):**
This mode starts monerod daemon as a local node, and automatically configures a public bootstrap node.
![image](https://user-images.githubusercontent.com/45968869/127621154-08ca9281-759f-47b8-afc2-bc27d9335449.png)
![image](https://user-images.githubusercontent.com/45968869/127359994-e0cb16cd-0ee1-4f5e-97ce-fa4b5e4c99d3.png)

**Advanced mode:**
Multiple options

Changes in NetworkStatusItem:
- remove "Network status" title
- display details about connected remote node. Display "Trusted remote node" or "Remote node" in advanced mode. Display "Public remote node" in simple mode.
- display whether SOCKS5 proxy is being used
- display whether bootstrap node is configured. Display "public bootstrap node" in simple mode (bootstrap). 
- "starting/stopping remote node seeker" status message in simple mode
- "switch remote node" button smaller, moved to the second line, and hidden when starting/stopping daemon
- use disconnected color for lightning icon when starting/stopping daemon

Changes in "Waiting for local node to stop..." processing splash:
- always display processing splash when closing Monero GUI (it was not displaying in simple mode (bootstrap))
- use message "Waiting for remote node seeker to stop..." in simple mode
- use message "Waiting for local node to stop..." in simple mode (bootstrap) and advanced mode
