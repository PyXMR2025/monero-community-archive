# Original Description
Bitmain has officially announced the Antminer X9, a dedicated miner targeting the RandomX algorithm, with shipments scheduled for July 2026.


**Key Specifications (Antminer X9)**

Based on Bitmain's official product listing:

- Hashrate: 1 MH/s (1000 KH/s) on RandomX

- Power Consumption: 2472 W

- Efficiency: ≈2.47 J/KH

- Price: USD 5,600 per unit

Bitmain claims the device utilizes custom RISC-V cores specifically optimized for RandomX. Relative to high-end consumer CPUs (e.g., Ryzen 9 class hardware, achieving ≈8−10 KH/s), this represents an estimated performance advantage per unit.

If widely adopted, large-scale deployment of these units could lead to significant hashrate concentration under Bitmain-controlled farms, undermining Monero's core principle of equitable, decentralized mining.

Bitmain Product Page (Antminer X9): https://shop.bitmain.com/product/detail?pid=00020251226185131923DUjfvPIO06BC
