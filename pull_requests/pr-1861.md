# Original Description
This PR adds a config file option to the wallets.

This is immensely useful, especially for server deployments. You can simply set all your configuration in the file (such as rpc port and credentials) and then start up your wallet.

This also increases security in terms of hiding configuration values from the rest of the system (which would show up in process listings and histories if given as command line parameters).
