# Original Description
This is a PR in my 'upstreaming seraphis_lib project', the changes here are not used anywhere yet.

- Save data points in csv format.
- Support incremental saving.

### Usage

In test `init()` methods:
```
if (params.core_params.td.get())
{
    TimingsDatabase::instance null_instance;
    null_instance.npoints = 0;

    std::string report_csv;
    std::string separator{','};
    report_csv += std::to_string(param_a) + separator;
    report_csv += std::to_string(param_b) + separator;

    params.core_params.td->add(report_csv.c_str(), null_instance);
}
```

In `tests/performance_tests/main.cpp`:
```
TEST_PERFORMANCE0(filter, p, test_1);
TEST_PERFORMANCE0(filter, p, test_2);

if (p.core_params.td.get())
  p.core_params.td->save(false);
```

Output recorded in file:
```
param_a,param_b,{comma-separated timings database stats}  //test_1
param_a,param_b,{comma-separated timings database stats}  //test_2
```
