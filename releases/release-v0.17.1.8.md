# Version: v0.17.1.8

# Release Notes
# Overview

This is the v0.17.1.8 minor point release of the Monero GUI software.
This is a recommended release that contains mitigations against the ongoing memory exhaustion attack.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.17.1.8)

Some highlights of this minor release are:

- Update monero submodule to v0.17.1.8
- UI tweaks to LineEdit component
- Minor bug fixes
