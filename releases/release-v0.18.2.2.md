# Version: v0.18.2.2

# Release Notes
# Overview

This is the v0.18.2.2 release of the Monero GUI software. This release adds support for Ledger Stax.

[The latest CLI release notes and downloads can be found on the release page here.](https://github.com/monero-project/monero/releases/tag/v0.18.2.2)

Some highlights of this release are:

- Add support for Ledger Stax (#4130)
- Update p2pool to v3.2 (#4142)
- Minor bug fixes and UI improvements
