# Original Description
There's an issue. It's a decently large issue. The issue is, that the current website isn't designed well, and the content is thrown wheverever it will fit without any thought put into how it can be the most easily accessible.

I have made two sets of redesign options, which can be viewed on the following albums: 

Redesign proposal 1 (drafts 1-3): http://imgur.com/a/MwyxX
Redesign proposal 2 (drafts 1-2): http://imgur.com/a/H9i3z

I think it's important we restructure the site and give it a makeover of some sort. A proposal has been made on the FFS which can be viewed here: https://forum.getmonero.org/6/ideas/87558/getmonero-com-redesign

I'm available to contact with any questions.
