# Original Description
It's almost 2022, so the copyright dates need to be changed once again. I saw a lot of PR's related to updating copyright to 2021, but most of them changed a different number of files. I was wondering if it would be easier if we simply moved to a yearless copyright, 
for example: `Copyright (c)  The Monero Project` .
 Or something like, `Copyright (c) 2014-present, The Monero Project`.
I think these options would require less maintenance. Thoughts?
