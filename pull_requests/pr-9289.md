# Original Description
Instead of hard-coding all missing key images into the code and inserting if missing, we simply pop all blocks after height 202611, forcing the node to re-sync. This PR makes the `monerod` binary about 0.3% smaller and only screws over people who are synced to height at least 202612 but haven't run a new binary in over 8 years with the DB fixup commit a98e976f9eabf2112a75347147becc1956ef8d66.
